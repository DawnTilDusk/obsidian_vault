## 1. 字符串函数全家桶
### 1. 构造与填充（打印图形/初始化）

有时候题目要求输出 20 个星号或者重复某个字符，不需要写循环：

- **重复字符构造**：`string s(10, 'a');` // 生成 "aaaaaaaaaa"
- **截取部分构造**：`string s2(s1, 2, 5);` // 从 s1 的下标 2 开始，截取 5 个字符构造新字符串
### 2. 插入、删除与替换（模拟题神器）

在处理复杂的字符串变换（如：把所有 "apple" 换成 "orange"）时极其有用：

- **删除**：`s.erase(pos, len);` // 从 pos 位置开始，删除 len 个字符。
- **插入**：`s.insert(pos, "content");` // 在 pos 位置插入字符串。
- **替换**：`s.replace(pos, len, "new_str");` // 把从 pos 开始的 len 个字符替换成新串。
- **清空**：`s.clear();` // 快速重置，用于多组数据输入。

### 3. 元素访问与边界（防止溢出）

- **首尾字符**：`s.front()` 和 `s.back()`（比写 `s[0]` 和 `s[s.size()-1]` 快且稳）。
- **判空**：`if (s.empty())` 优于 `if (s.size() == 0)`。
- **尾部增删**：`s.push_back('c');` 和 `s.pop_back();`（把 string 当栈用）。

### 4. 终极解析武器：`stringstream`

这是应对**“一行内有不确定数量的数字/单词”**或者**“读入一行乱码需要提取信息”**的绝对王牌。需要头文件 `#include <sstream>`。
```cpp
string line = "apple 123 orange 456";
stringstream ss(line);
string word;
int num;

while (ss >> word >> num) {
    // 第一次循环: word="apple", num=123
    // 第二次循环: word="orange", num=456
    cout << word << " owns " << num << endl;
}
```

> **考场场景**：如果题目给你一行文字，里面夹杂着空格、数字和单词，你先用 `getline` 读入整行，再扔进 `stringstream` 里慢慢 `>>` 出来。

#### 1. 核心操作：`clear()` 与 `str()` 的配合

如果你定义了一个全局的或在循环外的 `stringstream ss;`，在处理每一行新数据时，必须执行这两步：
```cpp
string line;
stringstream ss;
while (getline(cin, line)) {
    // 1. 重置流的状态标志（如 EOF 标志）
    ss.clear(); 
    // 2. 将新的一行内容放入流中
    ss.str(line); 

    int num;
    while (ss >> num) {
        // 处理数据
    }
}
```
> 注意： 很多同学只写 ss.str(line) 而忘记 ss.clear()。
> 原理： 当第一行读到末尾时，ss 会进入 EOF（End of File）状态。如果不调用 clear()，ss 会一直认为自己已经结束了，后续即使你塞入新字符串，ss >> num 也会直接失败。

---

#### 2. 更稳妥的写法：局部变量法

如果你担心记不住 `clear()`，最简单、最鲁棒的方法是**在循环内部定义 `stringstream`**。这样每次循环都会创建一个全新的对象，自动规避状态残留问题。
```cpp
while (getline(cin, line)) {
    stringstream ss(line); // 每次都是新的，不需要 clear()
    int num;
    while (ss >> num) {
        // ...
    }
}
```
---
#### 3. 处理“第一行是 N”的情况（重要陷阱）

计概 B 的题目通常第一行给出一个整数 $N$ 表示测试例数量。这时候如果你混合使用 `cin >> n` 和 `getline`，会触发之前提到的**换行符残留问题**。
**错误示范：**
```cpp
int n;
cin >> n; // 读完 n，换行符还在缓冲区
while(n--) {
    getline(cin, line); // 这里会读到一个空行！
    stringstream ss(line);
}
```
**正确示范：**
```cpp
int n;
cin >> n;
cin.ignore(); // 必须吃掉 n 后面的那个换行符！
// 或者使用 scanf("%d\n", &n); 这种写法也能吸收换行
while(n--) {
    getline(cin, line);
    // ...
}
```
---
#### 4. 复杂场景：一行内数据类型混杂
如果一行数据格式是 `ID:101 Name:Jack Score:95`，且你想提取数字，可以结合我们之前的“空格替换法”：
```
getline(cin, line);
for(char &c : line) if(!isdigit(c)) c = ' '; // 只要数字
stringstream ss(line);
int id, score;
ss >> id >> score; // 直接提取出 101 和 95
```
### 5. 常用算法配套（需 `#include <algorithm>`）
`string` 是可以像 `vector` 一样直接用 STL 算法的：

- **翻转**：`reverse(s.begin(), s.end());` // 处理回文串必用
- **排序**：`sort(s.begin(), s.end());` // 得到字符的字典序最小排列
- **去重**：
```cpp
	sort(s.begin(), s.end()); // 必须先排序
	s.erase(unique(s.begin(), s.end()), s.end()); // 此时 s 变成唯一字符集合
```

### 6. 查找的细节：`npos`

查找函数 `s.find()` 如果找不到，返回的是 `string::npos`。
```cpp
if (s.find("target") != string::npos) {
    // 找到了
} else {
    // 没找到
}
```

### 7. 其他函数

注意：`std::string` 相关函数在 `<string>` 中，字符判断在 `<cctype>`，C 风格字符串操作在 `<cstring>`。
#### `<string>` (针对 `std::string`)

- **转换：** `stoi(s)`, `stoll(s)`, `stod(s)` (字符串转 int/long long/double)。
- **反向转换：** `to_string(val)` (数字转字符串)。
- **查找：** `s.find("abc")` (返回下标，找不到返回 `string::npos`)；`s.find_last_of('/')` (找最后一个出现的字符)。
- **子串：** `s.substr(pos, len)` (从 pos 开始截取 len 长度)。
- **统计/修改：** 
	- `count(s.begin(), s.end(), 'a')` (需 `<algorithm>`)。
    - `reverse(s.begin(), s.end())` (翻转字符串)。
    - `sort(s.begin(), s.end())` (字符排序)。

#### `<cctype>` (单字符判断与转换)

- `isdigit(c)`, `isalpha(c)`, `isalnum(c)` (是否为数字/字母/数字字母)。
- `islower(c)`, `isupper(c)` (是否为小/大写)。
- `tolower(c)`, `toupper(c)` (转小/大写)。

## 2. 读入技巧与“换行符坑”

### 字符数组 (`char a[]`) vs `std::string`

- **字符数组读入：**
    - `scanf("%s", a)`：遇到空格、换行停止。
    - `cin.getline(a, size)`：读入一行，直到换行符，**会消耗掉换行符**。
- **`std::string` 读入：**
    - `cin >> s`：遇到空格、换行停止。
    - `getline(cin, s)`：读入一行，直到换行符，**会消耗掉换行符**。

> **核心陷阱：** 如果你先用 `cin >> n` 或 `scanf("%d", &n)` 读了一个数字，接着想用 `getline` 读一行，**必须**在中间加一个 `cin.ignore()` 或 `getchar()` 来吃掉那个数字后面的换行符。

---
## 3. scanf 文件读入与格式化

- **判断结束：** `while (scanf("%d", &n) != EOF)` 或 `while (cin >> n)`。
- **返回值：** `scanf` 返回成功读入的变量个数。如果格式不匹配，返回 0。
- **处理特殊格式：**

    - 跳过特定字符：`scanf("%d-%d-%d", &y, &m, &d);` (自动过滤中间的减号)。
    - 限制读入长度：`scanf("%4d%2d%2d", &y, &m, &d);` (处理 `20231225` 这种紧凑格式)。
    - 读入整行到字符数组：`scanf("%[^\n]", str);` (不推荐，建议用 `getline`)。

## 4. 日期模拟通用模板

```cpp
int daysOfMonth[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

bool isLeap(int y) {
    return (y % 4 == 0 && y % 100 != 0) || (y % 400 == 0);
}

// 方法一：万能解析 (支持 2023-12-01, 2023/12/01, 20231201)
void parseDate(string s, int &y, int &m, int &d) {
    if (s.find('-') != string::npos || s.find('/') != string::npos) {
        for (auto &c : s) if (c == '-' || c == '/') c = ' '; // 替换分隔符为空格
        stringstream ss(s);
        ss >> y >> m >> d;
    } else { // 紧凑型 yyyymmdd
        y = stoi(s.substr(0, 4));
        m = stoi(s.substr(4, 2));
        d = stoi(s.substr(6, 2));
    }
}

// 方法二：日期加法
void addDays(int &y, int &m, int &d, int step) {
    for (int i = 0; i < step; i++) {
        daysOfMonth[2] = isLeap(y) ? 29 : 28;
        d++;
        if (d > daysOfMonth[m]) {
            d = 1; m++;
            if (m > 12) { m = 1; y++; }
        }
    }
}
```

```cpp
/*恶心的读入
3
2022-01-01 12:00:00 UTC-5
2023-06-01 23:59:59 UTC-8
2024-03-15 00:00:00 UTC+10
*/

#include <iostream>
#include <string>
#include <cstdio>
#include <vector>

using namespace std;

int daysOfMonth[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

bool isLeap(int y) {
    return (y % 4 == 0 && y % 100 != 0) || (y % 400 == 0);
}

// 核心函数：处理时间进位/退位
void updateTime(int &y, int &m, int &d, int &h, int offset) {
    h += offset;
    // 处理小时向天的进位
    while (h >= 24) {
        h -= 24;
        d++;
        daysOfMonth[2] = isLeap(y) ? 29 : 28;
        if (d > daysOfMonth[m]) {
            d = 1; m++;
            if (m > 12) { m = 1; y++; }
        }
    }
    // 处理小时向天的退位 (针对 offset 为负的情况)
    while (h < 0) {
        h += 24;
        d--;
        if (d < 1) {
            m--;
            if (m < 1) { m = 12; y--; }
            daysOfMonth[2] = isLeap(y) ? 29 : 28;
            d = daysOfMonth[m];
        }
    }
}

int main() {
    int n;
    scanf("%d", &n);
    while (n--) {
        int y, m, d, hh, mm, ss, x;
        char tz[15];
        // 1. 利用格式化读入处理 YYYY-MM-DD HH:MM:SS
        scanf("%d-%d-%d %d:%d:%d", &y, &m, &d, &hh, &mm, &ss);
        // 2. 读入 UTC 字符串
        scanf("%s", tz); // 读入类似 "UTC-5" 或 "UTC+10"
        // 3. 解析时区数字
        string s = tz;
        if (s.find('+') != string::npos) {
            x = stoi(s.substr(s.find('+') + 1));
        } else {
            x = -stoi(s.substr(s.find('-') + 1));
        }
        // 4. 计算偏移并更新
        updateTime(y, m, d, hh, 8 - x);
        // 5. 格式化输出，注意补零 (%02d)
        printf("%04d-%02d-%02d %02d:%02d:%02d\n", y, m, d, hh, mm, ss);
    }
    return 0;
}
```

## 5. 字符串 Hash 与结构体排序

面对“名字”转“ID”的需求，最快的方法是使用 `std::map`。

```cpp
#include <map>
#include <vector>
#include <algorithm>
using namespace std;

map<string, int> nameToId;
int currentId = 0;

int getId(string name) {
    if (nameToId.find(name) == nameToId.end()) {
        nameToId[name] = ++currentId;
    }
    return nameToId[name];
}

struct Team {
    string name;
    int score = 0;
    // 自定义排序规则
    bool operator<(const Team& other) const {
        if (score != other.score) return score > other.score; // 分高在前
        return name < other.name; // 字典序
    }
} teams[105];
```

## 6. 基础算法模板

### 二分查找 (查找第一个满足条件的 x)
```c++
int l = 0, r = max_val, ans = 0;
while (l <= r) {
    int mid = l + (r - l) / 2;
    if (check(mid)) {
        ans = mid; // 记录答案
        r = mid - 1; // 尝试更小的
    } else l = mid + 1;
}
```
### DFS 框架
```cpp
void dfs(int step, ...) {
    if (到达终点) { 更新答案; return; }
    for (每一种可能的选择) {
        if (合法) {
            标记访问;
            dfs(step + 1, ...);
            取消标记 (回溯);
        }
    }
}
```
### BFS 框架
```cpp
queue<int> q;
q.push(start);
vis[start] = 1;
while (!q.empty()) {
    int cur = q.front(); q.pop();
    for (所有相邻节点 next) {
        if (!vis[next]) {
            vis[next] = 1;
            dist[next] = dist[cur] + 1;
            q.push(next);
        }
    }
}
```

## 7. 算法常见模型总结

- **DFS (深度优先搜索)：** 全排列问题、走迷宫（找所有路径）、八皇后、组合问题。
- **BFS (广度优先搜索)：** 最短路径（步数最少）、层序遍历。
- **简单 DP：** * **背包问题：** 选或不选，价值最大。
    - **LIS (最长上升子序列)：** 状态 $dp[i]$ 表示以 $i$ 结尾的最长长度。
    - **LCS (最长公共子序列)：** $dp[i][j]$ 比较两个字符串。
- **简单贪心：** 区间调度（按结束时间排序）、荷马复仇（哈夫曼树）、硬币找零（特定面额）。
- **二分答案**
- **素数筛/欧拉筛**

### 一、 DFS (深度优先搜索)

#### 1. 全排列 (Permutation)
如果题目只是要求字典序全排列，直接用 `next_permutation`。如果需要手动实现（如添加额外限制）：
```cpp
int n, a[15], vis[15], path[15];
void dfs(int step) {
    if (step == n + 1) {
        for (int i = 1; i <= n; i++) printf("%d ", path[i]);
        puts(""); return;
    }
    for (int i = 1; i <= n; i++) {
        if (!vis[i]) {
            vis[i] = 1; path[step] = a[i];
            dfs(step + 1);
            vis[i] = 0; // 回溯
        }
    }
}
```
#### 2. 走迷宫 (Find All Paths / Existence)
```cpp
int dx[] = {0, 0, 1, -1}, dy[] = {1, -1, 0, 0};
void dfs(int x, int y) {
    if (x == tx && y == ty) { ans++; return; }
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i], ny = y + dy[i];
        if (nx >= 1 && nx <= n && ny >= 1 && ny <= m && !vis[nx][ny] && map[nx][ny] != '#') {
            vis[nx][ny] = 1;
            dfs(nx, ny);
            vis[nx][ny] = 0;
        }
    }
}
```
#### 3. 八皇后 (N-Queens)
```cpp
int col[15], dg[30], udg[30]; // 列、正对角线、反对角线
void dfs(int r) {
    if (r == n + 1) { ans++; return; }
    for (int c = 1; c <= n; c++) {
        if (!col[c] && !dg[r + c] && !udg[r - c + n]) {
            col[c] = dg[r + c] = udg[r - c + n] = 1;
            dfs(r + 1);
            col[c] = dg[r + c] = udg[r - c + n] = 0;
        }
    }
}
```

### 二、 BFS (广度优先搜索)
#### 1. 最短路径 (2D Grid)
```cpp
struct Node { int x, y, dist; };
int bfs() {
    queue<Node> q;
    q.push({sx, sy, 0});
    vis[sx][sy] = 1;
    while (!q.empty()) {
        Node cur = q.front(); q.pop();
        if (cur.x == tx && cur.y == ty) return cur.dist;
        for (int i = 0; i < 4; i++) {
            int nx = cur.x + dx[i], ny = cur.y + dy[i];
            if (nx >= 1 && nx <= n && ny >= 1 && ny <= m && !vis[nx][ny] && g[nx][ny] != '#') {
                vis[nx][ny] = 1;
                q.push({nx, ny, cur.dist + 1});
            }
        }
    }
    return -1; // 不通
}
```

### 三、 简单 DP (动态规划)
#### 1. 0/1 背包 (一维优化)
```cpp
int dp[1005]; // dp[j] 表示容量为 j 时的最大价值
for (int i = 1; i <= n; i++) {
    for (int j = m; j >= w[i]; j--) { // 逆序是为了保证每件物品只选一次
        dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    }
}
```
#### 2. LIS (最长上升子序列 - $O(n^2)$ 版)
```cpp
int dp[1005], a[1005], ans = 0;
for (int i = 1; i <= n; i++) {
    dp[i] = 1;
    for (int j = 1; j < i; j++) {
        if (a[j] < a[i]) dp[i] = max(dp[i], dp[j] + 1);
    }
    ans = max(ans, dp[i]);
}
```
#### 3. LCS (最长公共子序列)
```cpp
int dp[1005][1005];
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        if (s1[i-1] == s2[j-1]) dp[i][j] = dp[i-1][j-1] + 1;
        else dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
    }
}
```

### 四、 简单贪心 (Greedy)
#### 1. 区间调度 (最多参加多少个活动)
**策略：按结束时间从早到晚排序。**
```cpp
struct Interval { int s, e; } arr[1005];
bool cmp(Interval a, Interval b) { return a.e < b.e; }

sort(arr, arr + n, cmp);
int cnt = 0, last_e = -1;
for (int i = 0; i < n; i++) {
    if (arr[i].s >= last_e) {
        cnt++;
        last_e = arr[i].e;
    }
}
```
#### 2. 荷马复仇 (Huffman 树/合并果子)(小根堆模板题)
**策略：每次合并最小的两个。**
>[!NOTE]
>- `priority_queue<int> q;` → **大**根堆（大的先出）。
>- `priority_queue<int, vector<int>, greater<int>> q;` → **小**根堆（小的先出）。

```cpp
priority_queue<int, vector<int>, greater<int>> pq; // 小根堆
for (int i = 0; i < n; i++) pq.push(a[i]);
long long total_cost = 0;
while (pq.size() > 1) {
    int x = pq.top(); pq.pop();
    int y = pq.top(); pq.pop();
    total_cost += (x + y);
    pq.push(x + y);
}
```
#### 3. 硬币找零 (简单贪心版)
**注意：仅在面额成倍数关系或特定情况下成立，否则需用 DP。**
```cpp
int coins[] = {100, 50, 20, 10, 5, 1}, money = 286;
for (int i = 0; i < 6; i++) {
    cnt += money / coins[i];
    money %= coins[i];
}
```

### 五、二分答案
### 1. 题干的核心标志词

这是判断是否用二分答案的最直接依据：
- **求最小值最大化**：例如“安排 $N$ 头牛进 $M$ 个隔间，使最近的两头牛之间的距离**最大**”。
- **求最大值最小化**：例如“将一串任务分成 $K$ 组，求所有组中工作量最大的那一组的**最小值**”。
- **求满足条件的极值**：例如“切绳子，要求切出至少 $K$ 段，求每段长度的**最大值**”。

---
### 2. 为什么能用二分？（单调性判断）

要用二分答案，问题必须满足**单调性**。
- **逻辑**：如果长度为 $x$ 时可以切出 $K$ 段，那么长度为 $x-1$ 时一定也能切出 $K$ 段（甚至更多）。
- **分界点**：存在一个临界值，比它小的都“可行”，比它大的都“不可行”。我们要找的就是这个临界点。

---
### 3. 三大经典题型具体化

#### A. 锯木头/切绳子类（求最大长度）
- **题干**：有 $N$ 根长度不一的木头，要切出 $K$ 段长度均为 $L$ 的小木块。求 $L$ 的最大值。
- **`check(mid)` 逻辑**：遍历 $N$ 根木头，计算 $\sum (\text{length}_i / mid)$，看总量是否 $\ge K$。
#### B. 摊派/分组类（求最小负荷）
- **题干**：有 $N$ 本书要分给 $M$ 个人抄写，每本书页数不同。要求每个人分配到的书必须是连续的。求抄写页数最多的人所抄页数的最小值。
- **`check(mid)` 逻辑**：尝试以 $mid$ 为上限给每个人分配任务，看 $M$ 个人够不够用。
#### C. 进击的奶牛/丢瓶盖类（求最大间距）
- **题干**：在一条直线上有 $N$ 个坐标点，选出 $C$ 个点，使得任意两点间距离的最小值最大。
- **`check(mid)` 逻辑**：贪心放置。第一个点必放，然后找距离上一个点 $\ge mid$ 的最近点放下一个，看最后能不能放下 $C$ 个。
---
### 4. 考场代码模板（防死循环版）
二分最怕死循环。建议统一使用 `l <= r` 的写法，这种写法最符合直觉：
```cpp
long long l = min_possible, r = max_possible;
long long ans = l;

while (l <= r) {
    long long mid = l + (r - l) / 2; // 防止加法溢出
    if (check(mid)) {
        ans = mid;    // 记录下这个可行解
        l = mid + 1;  // 既然可行，尝试找更大的（如果是求最大值最大化）
    } else {
        r = mid - 1;  // 不可行，只能往小了找
    }
}
cout << ans << endl;
```
---
### 5. 易错点提示
1. **边界范围 (`l` 和 `r`)**：
    - `l` 常常取 $0$ 或 $1$。
    - `r` 往往取题目给出的最大范围（如 $10^9$），或者数组中所有元素的总和。**注意 `r` 是否需要开 `long long`**。
2. **`check` 函数的复杂度**：
    - `check` 函数内部通常是一个 $O(N)$ 的遍历。
    - 总复杂度是 $O(N \log (\text{Range}))$，对于 $N=10^5$ 的数据量非常轻松。
3. **精度问题**：
    - 如果题目要求输出浮点数（如保留两位小数），循环条件应改为 `for(int i=0; i<100; i++)`（强制循环100次提高精度）或 `while(r - l > 1e-7)`。

### 六、素数筛/欧拉筛
筛素数`prime[i]`
```cpp
const int MAXN = 1000005; // 根据题目要求调整大小
int primes[MAXN];         // 存储发现的素数
bool is_not_prime[MAXN];  // 标记是否为合数，false表示是素数
int cnt = 0;              // 记录素数个数

void get_primes(int n) {
    is_not_prime[0] = is_not_prime[1] = true; // 0和1不是素数
    for (int i = 2; i <= n; i++) {
        if (!is_not_prime[i]) {
            primes[cnt++] = i; // 如果没被筛掉，就是素数
        }
        // 核心：遍历已找到的素数
        for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
            is_not_prime[i * primes[j]] = true; // 筛掉合数
            // 关键点：如果 i 能被某个质数整除，停止
            // 这保证了 i * primes[j+1] 会被后面的更大i筛掉
            if (i % primes[j] == 0) break; 
        }
    }
}
```
筛欧拉函数`phi[i]`
```cpp
const int MAXN = 1000005;
int primes[MAXN], phi[MAXN], cnt = 0;
bool is_not_prime[MAXN];

void get_phi(int n) {
    phi[1] = 1; // 特判：1的欧拉函数是1
    for (int i = 2; i <= n; i++) {
        if (!is_not_prime[i]) {
            primes[cnt++] = i;
            phi[i] = i - 1; // 性质1：i是素数
        }
        for (int j = 0; j < cnt && i * primes[j] <= n; j++) {
            is_not_prime[i * primes[j]] = true;
            
            if (i % primes[j] == 0) {
                // 性质2：primes[j]是i的因子
                phi[i * primes[j]] = phi[i] * primes[j]; 
                break; 
            } else {
                // 性质3：i与primes[j]互质
                phi[i * primes[j]] = phi[i] * (primes[j] - 1);
            }
        }
    }
}
```
## 8.高精度运算

在考场上，建议使用 `vector<int>` 来存储数字，**低位在前，高位在后**（即 `a[0]` 存个位）。这样进位时只需 `push_back`，处理起来最方便。

---
### 1. 结构准备与读入
```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

// 将字符串转为 vector<int>，个位在 index 0
vector<int> read(string s) {
    vector<int> a;
    for (int i = s.size() - 1; i >= 0; i--) a.push_back(s[i] - '0');
    return a;
}

// 打印函数
void print(vector<int> a) {
    for (int i = a.size() - 1; i >= 0; i--) printf("%d", a[i]);
    puts("");
}
```
---
### 2. 高精度加法 (A + B)
**逻辑：** 逐位相加，处理进位。
```cpp
vector<int> add(vector<int> A, vector<int> B) {
    vector<int> C;
    int t = 0; // 进位
    for (int i = 0; i < A.size() || i < B.size(); i++) {
        if (i < A.size()) t += A[i];
        if (i < B.size()) t += B[i];
        C.push_back(t % 10);
        t /= 10;
    }
    if (t) C.push_back(t);
    return C;
}
```

### 3. 高精度减法 (A - B)
**逻辑：** 逐位相减，借位。前提需保证 $A \ge B$。
```cpp
// 辅助：判断是否 A >= B
bool cmp(vector<int> A, vector<int> B) {
    if (A.size() != B.size()) return A.size() > B.size();
    for (int i = A.size() - 1; i >= 0; i--)
        if (A[i] != B[i]) return A[i] > B[i];
    return true;
}

vector<int> sub(vector<int> A, vector<int> B) {
    vector<int> C;
    int t = 0; // 借位
    for (int i = 0; i < A.size(); i++) {
        t = A[i] - t;
        if (i < B.size()) t -= B[i];
        C.push_back((t + 10) % 10); // 合并处理正负情况
        if (t < 0) t = 1; else t = 0;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back(); // 去除前导0
    return C;
}
```

### 4. 高精度乘法 (高精 A × 低精 b)
**逻辑：** 用低精数字去乘高精数字的每一位。
```cpp
vector<int> mul(vector<int> A, int b) {
    vector<int> C;
    if (b == 0) return {0};
    int t = 0;
    for (int i = 0; i < A.size() || t; i++) {
        if (i < A.size()) t += A[i] * b;
        C.push_back(t % 10);
        t /= 10;
    }
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```

### 5. 高精度除法 (高精 A ÷ 低精 b)
**逻辑：** 从高位往低位除，余数留给下一位。
```cpp
vector<int> div(vector<int> A, int b, int &r) { // r是余数
    vector<int> C;
    r = 0;
    for (int i = A.size() - 1; i >= 0; i--) {
        r = r * 10 + A[i];
        C.push_back(r / b);
        r %= b;
    }
    reverse(C.begin(), C.end()); // 结果为了统一是个位在前，需要翻转
    while (C.size() > 1 && C.back() == 0) C.pop_back();
    return C;
}
```
