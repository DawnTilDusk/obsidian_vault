# 📚 函数凹凸性学习笔记 (单变量)

## 📌 核心定义与等价判定

在微积分中，我们通常将“开口向上”的函数称为**凸函数**（Convex Function），将“开口向下”的函数称为**凹函数**（Concave Function）。

### 1. 凸函数 (Convex Function)

| **知识点**     | **代数表达式 (定义)**                                                                               | **几何意义**       | **证明思路**                               |
| ----------- | -------------------------------------------------------------------------------------------- | -------------- | -------------------------------------- |
| **定义 (零阶)** | $f(\theta x_1 + (1-\theta)x_2) \le \theta f(x_1) + (1-\theta)f(x_2)$，其中 $\theta \in [0, 1]$。 | **割线在函数图像上方。** | 几何直观，所有高级性质的起点。                        |
| **一阶判定**    | $f(x) \ge f(x_0) + f'(x_0)(x - x_0)$                                                         | **切线在函数图像下方。** | 泰勒展开（一阶），是连接导数和函数值的桥梁。                 |
| **二阶判定**    | $f''(x) \ge 0$                                                                               | **曲率非负，开口向上。** | 利用一阶判定：$f'(x)$ 单增 $\iff f''(x) \ge 0$。 |

### 2. 凹函数 (Concave Function)

| **知识点**  | **代数表达式 (定义)**                                                       | **几何意义**       | **关联性**                            |
| -------- | -------------------------------------------------------------------- | -------------- | ---------------------------------- |
| **定义**   | $f(\theta x_1 + (1-\theta)x_2) \ge \theta f(x_1) + (1-\theta)f(x_2)$ | **割线在函数图像下方。** | **$f(x)$ 是凹函数 $\iff -f(x)$ 是凸函数。** |
| **一阶判定** | $f(x) \le f(x_0) + f'(x_0)(x - x_0)$                                 | **切线在函数图像上方。** | $\ln x$ 是凹函数的典型例子。                 |
| **二阶判定** | $f''(x) \le 0$                                                       | **曲率非正，开口向下。** | 导数 $f'(x)$ 单调递减。                   |

## 🚀 一级结论：微分性质 (等价条件)

一级结论是基于函数可微性导出的，是期末考中最常用的判定和证明工具。

### 一、斜率关系
#### 1. 割线斜率单调性 (三弦引理)

>[!NOTE]
>**想象内接三角形，三边斜率关系一定明确**

- 结论：对于凸函数 $f(x)$，任取 $x_1 < x_2 < x_3$，则割线斜率递增：
$$k_{12} \le k_{13} \le k_{23}$$
- **证明思路**：利用凸函数定义，将中间点 $x_2$ 写成 $x_1$ 和 $x_3$ 的加权平均，进行代数变形即可得证。

- **关联性**：这是**零阶定义直接推导**，是后续所有单调性结论的基础。

#### 2. 固定一端点的割线斜率单调性

- **结论**：固定左端点 $x_1$，割线斜率 $k(x) = \frac{f(x) - f(x_1)}{x - x_1}$ 随着 $x$ 的增大而**单调递增**。

- **证明思路**：直接利用三弦引理 $k_{13} \le k_{12}$（其中 $x_3 < x_2$），证明 $\varphi(x)$ 是 $x$ 的单调递增函数。这是证明单侧导数存在性的基础。
#### 3. 四点六弦斜率关系 (推广)

- **结论**：对于凸函数 $f(x)$ 上的四个点 $x_1 < x_2 < x_3 < x_4$，它们构成的六条割线斜率 $k_{ij}$ 满足以下核心关系：
    1. 局部递增链（最强单调性）：
$$k_{12} \le k_{23} \le k_{34}$$
    2. 全局递增链（由固定端点原理得出）：
$$k_{12} \le k_{13} \le k_{14} \le k_{24} \le k_{34}$$
- **证明思路**：
    1. **递增链**：任取其中三个点，应用三次三弦引理即可组合出上述链式不等式。

    2. **局部与全局的不确定性**：斜率 $k_{23}$（局部）和 $k_{14}$（全局）**无法直接比较**。可以通过构造 $f(x)=x^2$ 的反例来证明两者大小关系不确定，这取决于 $x_i$ 的间隔分布。

- **关联性**：展示了凸函数斜率单调性在复杂组合下的应用，同时也揭示了**几何限制**（即仅凭凸性无法确定所有割线的大小关系）。
### 二、导数关系
#### 1. 单侧导数存在性与关系

- 结论：凸函数在开区间内处处存在单侧导数，且在任意一点 $x$ 满足：$$f'\_(x) \le f'\_+(x)$$
- **证明思路**：
    - 证明割线斜率函数 $\varphi(h)$ **单调递增**（通过三弦引理）且**有下界**。
    - 根据**单调有界定理**，极限（即右导数）存在。左导数对称可证。

- **关联性**：解决了**不可导点**（如尖点）的凸性判断问题。

#### 2. 单侧导数的单调性 (广义导数单调性)

- 结论：对于凸函数 $f(x)$，任取两点 $x_1 < x_2$，则 $x_1$ 处的右导数总是小于或等于 $x_2$ 处的左导数：
$$f'\_+(x_1) \le f'\_-(x_2)$$
- **证明思路**：
    - **核心**：利用 $x_1 < x_2$ 之间的任意割线斜率 $k_{12} = \frac{f(x_2)-f(x_1)}{x_2-x_1}$。

    - **左侧极限**：取极限 $x \to x_1^+$，根据 $f'\_+(x_1)$ 的定义，有 $f'\_+(x_1) \le k_{12}$。
    - **右侧极限**：取极限 $x \to x_2^-$, 根据 $f'\_-(x_2)$ 的定义，有 $k_{12} \le f'\_-(x_2)$。

    - 结合两者即得 $f'\_+(x_1) \le k_{12} \le f'\_-(x_2)$。

- **关联性**：这是**导数单调性**的**最一般形式**。它涵盖了不可导函数，并保证了凸函数广义导数（次梯度）的单调性。

### 3. 导函数的单调性 (二阶判定基础)

- **结论**： $f(x)$ 是凸函数 
	- **（如果可导）** $\iff f'(x)$ 是单调递增函数 
	- **（如果二阶可导）**$\iff f''(x)$恒正

- **证明思路**：这是**单侧导数单调性**的特例。如果 $f(x)$ 可导，则 $f'(x) = f'\_+(x) = f'\_-(x)$，直接由上一个结论推导出 $f'(x_1) \le f'(x_2)$。
    
- **关联性**：直接连接了**二阶导数 $f''(x) \ge 0$** 的判定。

## 💡 二级结论：极值与不等式 (应用性质)

二级结论是由凸性推导出的，常用于解决优化和证明不等式的问题。

### 1. 局部极值与全局极值 (Local $\to$ Global)

- **结论**：如果 $f(x)$ 是凸函数，则它的**任意局部最小值点都是全局最小值点**。

- **证明思路**：
    - 反证法：假设存在一个更低的点 $y$。连接 $x_0$ 和 $y$ 的割线必须在函数图像上方。
    - 根据一阶判定 $f(y) \ge f(x_0) + f'(x_0)(y-x_0)$，如果 $f'(x_0)=0$，则 $f(y) \ge f(x_0)$，故 $x_0$ 是全局最小值。

- **关联性**：这是凸优化理论的核心，保证了数值方法（如牛顿法）能找到正确解。

### 2. 最大值在端点 (Maximum at Boundary)

- **结论**：凸函数在闭区间 $[a, b]$ 上的最大值一定在端点 $f(a)$ 或 $f(b)$ 处取得。
    
- **证明思路**：
    
    - 反证法：假设最大值 $f(x_c)$ 在 $x_c \in (a, b)$ 处取得。
    - 将 $x_c$ 写成 $x_c = \theta a + (1-\theta)b$，根据凸函数定义：
$$f(x_c) \le \theta f(a) + (1-\theta)f(b)$$
    - 由于 $\theta, 1-\theta > 0$，加权平均值 $\theta f(a) + (1-\theta)f(b)$ 绝不可能严格小于 $\max(f(a), f(b))$，更不可能严格小于 $f(x_c)$。因此 $f(x_c)$ 不可能严格大于 $\max(f(a), f(b))$。

- **关联性**：简化了约束优化问题中求最大值的步骤。

### 3. 琴生不等式 (Jensen's Inequality)

- 结论：对于凸函数 $f(x)$ 和 $\lambda_i \ge 0, \sum \lambda_i = 1$，有：
$$f\left(\sum \lambda_i x_i\right) \le \sum \lambda_i f(x_i)$$
- **证明思路**：**数学归纳法**。
    - $n=2$ 时即凸函数定义。
    - 假设 $n=k$ 成立，证明 $n=k+1$ 时，通过将 $k+1$ 项拆分成一项和剩余 $k$ 项的加权平均（进行两次 $n=2$ 的代换）即可完成证明。

- **关联性**：这是**凸性与平均值**关系的体现。

### 4. 赫尔德不等式 (Hölder's Inequality)

赫尔德不等式是**琴生不等式**在**指数函数凸性**上的一个重要应用，是柯西-施瓦茨不等式的推广，是现代分析学的基础工具。

- **前提条件**：存在一对共轭指数 $p > 1$ 和 $q > 1$，满足 $\frac{1}{p} + \frac{1}{q} = 1$。
- 离散形式（结论）：对于两组非负实数 $a_i, b_i$：$$\sum_{i=1}^n a_i b_i \le \left(\sum_{i=1}^n a_i^p\right)^{1/p} \left(\sum_{i=1}^n b_i^q\right)^{1/q}$$
- **证明思路**：
    
    1. **归一化**：定义 $A = \sum a_i^p$ 和 $B = \sum b_i^q$，构造 $\sum x_i = 1$ 和 $\sum y_i = 1$ 的新变量 $x_i = a_i^p / A$ 和 $y_i = b_i^q / B$。
    2. 核心不等式：利用凹函数 $\ln t$ 或凸函数 $e^t$ 的性质，推导出 Young 不等式：
$$x_i^{1/p} y_i^{1/q} \le \frac{1}{p} x_i + \frac{1}{q} y_i$$
    3. **求和收尾**：对 Young 不等式求和，利用 $\sum x_i = 1, \sum y_i = 1$ 和 $\frac{1}{p} + \frac{1}{q} = 1$，即可证得
$$\sum (x_i^{1/p} y_i^{1/q}) \le 1$$
    4. **反代换**：将结果代回原式 $\sum a_i b_i \le A^{1/p} B^{1/q} (1)$ 即得最终结论。

- **关联性**：
    - **应用**：当 $p=q=2$ 时，它退化为**柯西-施瓦茨不等式**。
    - **数学分析**：它是证明**闵可夫斯基不等式**（三角不等式在 $L^p$ 空间中的形式）的关键一步，是现代分析学中定义范数和空间的基础。

## ⚙️ 特殊应用与联系：牛顿切线法

牛顿切线法（寻找函数 $f(x)=0$ 的根）是一种迭代算法，其几何意义是**用切线来近似函数**。

### 1. 牛顿法迭代公式

- **目标**：寻找 $f(x)=0$ 的根 $x^*$。
- **几何迭代**：在当前点 $x_n$ 处作切线，切线与 $x$ 轴的交点即为下一次迭代点 $x_{n+1}$。
- 公式：$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$
### 2. 牛顿法与凸函数的联系

凸函数或凹函数能够极大地保证牛顿法的**收敛速度**和**稳定性**。

|**关系点**|**凸函数 f(x) 的保证**|**对牛顿法的影响**|
|---|---|---|
|**几何位置**|**切线在函数下方** (一阶判定)|如果 $f(x)$ 凸且 $f'(x) > 0$，则 $x_{n+1}$ 总是大于 $x^*$（位于根的右侧），反之亦然。这帮助我们理解迭代点的移动方向。|
|**优化保证**|**局部最优即全局最优**|**（在优化问题 $\min f(x)$ 中）** 凸性保证了牛顿法不会陷入局部极小值，迭代方向总是指向唯一的全局最小值。|
|**稳定性**|**$f''(x) \ge 0$**|保证了迭代公式中的分母（$f'(x_n)$）变化规律是单调的，避免了迭代过程中的剧烈震荡。|

### 3. 牛顿法的收敛性 (二次收敛)

- 结论：如果 $x^*$ 是方程 $f(x)=0$ 的单根（$f'(x^*) \neq 0$），且初始点足够接近 $x^*$，则牛顿法具有二次收敛速度。$$|e_{n+1}| \le C |e_n|^2$$（其中 $e_n = x_n - x^*$ 是误差，$C$ 是常数 $\approx |\frac{f''(x^*)}{2 f'(x^*)}|$）

- **证明思路 (二次收敛)**：
    1. **泰勒展开**：将 $f(x^*)$ 在 $x_n$ 处进行二阶泰勒展开，并利用 $f(x^*)=0$ 的性质。
    2. **代入公式**：将泰勒展开结果代入牛顿法迭代公式 $e_{n+1} = e_n - \frac{f(x_n)}{f'(x_n)}$ 中。
    3. **结论**：经过代数消去，得到 $e_{n+1} = \frac{f''(\xi_n)}{2 f'(x_n)} e_n^2$ 的关系式。

- **关联性**：凸性 ($f''(x) \ge 0$) 使常数 $C$ 的符号稳定，进一步确保了收敛的稳定性。二次收敛意味着迭代一次，有效数字的位数大约翻一倍，速度非常快。

## 🎯 典型例题与解题技巧

### 例题 1：判定凸性 (二阶导数法)

**题目**：判断函数 $f(x) = x \ln x$ 在 $(0, +\infty)$ 上的凹凸性。

- **解题步骤**：
    1. 求一阶导：$f'(x) = \ln x + x \cdot \frac{1}{x} = \ln x + 1$。
    2. 求二阶导：$f''(x) = \frac{1}{x}$。
    3. **判定**：在 $(0, +\infty)$ 上，$x > 0$，所以 $f''(x) = \frac{1}{x} > 0$。

- **结论**：$f(x) = x \ln x$ 是**凸函数**。

### 例题 2：证明不等式 (切线法)

**题目**：证明对于任意 $x \in \mathbb{R}$，有 $e^x \ge 1 + x$。

- **解题步骤**：
    
    1. 构造函数 $f(x) = e^x$。
    2. 判定凸性：$f'(x) = e^x, f''(x) = e^x > 0$。故 $f(x)$ 是**凸函数**。
    3. 利用一阶判定（切线不等式）：
$$f(x) \ge f(x_0) + f'(x_0)(x - x_0)$$
    4. 选择切点 $x_0 = 0$：
        - $f(0) = e^0 = 1$
        - $f'(0) = e^0 = 1$
        - 代入不等式：$e^x \ge 1 + 1 \cdot (x - 0)$

- **结论**：$e^x \ge 1 + x$ 得证。

### 例题 3：应用琴生不等式

**题目**：证明对于任意 $a, b > 0$，有 $\frac{a+b}{2} \ge \sqrt{ab}$（即算术平均数 $\ge$ 几何平均数）。

- **解题步骤**：
    1. **构造凹函数**：选取 $f(x) = \ln x$。
    2. 判定凹性：$f''(x) = -\frac{1}{x^2} < 0$，故 $f(x)$ 是**凹函数**。

    3. 应用琴生不等式（凹函数版本，符号相反 $\ge$）：
$$f\left(\sum \lambda_i x_i\right) \ge \sum \lambda_i f(x_i)$$
    4. 取 $x_1=a, x_2=b$，取 $\lambda_1 = \lambda_2 = 1/2$：$$\ln\left(\frac{a}{2} + \frac{b}{2}\right) \ge \frac{1}{2} \ln a + \frac{1}{2} \ln b$$
    5. 利用对数性质：
$$\ln\left(\frac{a+b}{2}\right) \ge \ln a^{\frac{1}{2}} + \ln b^{\frac{1}{2}} = \ln(a^{\frac{1}{2}} b^{\frac{1}{2}}) = \ln(\sqrt{ab})$$
    6. 结论：由于 $\ln x$ 是单调递增函数，可去掉对数：
$$\frac{a+b}{2} \ge \sqrt{ab}$$
