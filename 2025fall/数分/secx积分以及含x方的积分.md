## 💡 微积分规律整理与归纳

### 1. 三角函数积分的本质联系

我们发现 $\sec x$ 的积分结果在代数结构上与 $\frac{1}{1-x^2}$ 的积分结果存在深刻的平行关系。

- 核心直觉： $\int \sec x \, dx$ 可以通过巧妙的 $u = \sin x$ 换元进行简化。
$$\int \sec x \, dx = \int \frac{\cos x}{1-\sin^2 x} dx \xrightarrow{u=\sin x} \int \frac{1}{1-u^2} du$$
- 结果形式的等价性： 这表明 $\sec x$ 的积分本质上是反双曲正切函数 ($\text{artanh } u$) 与 三角函数 ($\sin x$) 的复合。这体现在两个等价的公式中：
$$\ln|\sec x + \tan x| \equiv \frac{1}{2}\ln\left|\frac{1+\sin x}{1-\sin x}\right|$$
- **规律：** 这揭示了三角函数和双曲函数之间存在着深层的代数对称性。

### 2. 双曲函数与反函数（对数形式）

绝大多数双曲函数的反函数都可以用对数形式表示，这为涉及 $\sqrt{x^2 \pm a^2}$ 的积分提供了简洁的解答。

- **规律：** 当积分中出现 $\sqrt{x^2 \pm a^2}$ 时，若不使用三角换元，通常可以直接得出其反双曲函数形式的解。

- **核心公式（$a=1$）：**
    - **反双曲正弦：** $\text{arsinh } x = \ln(x + \sqrt{x^2 + 1})$
    - **反双曲余弦：** $\text{arcosh } x = \ln(x + \sqrt{x^2 - 1})$
    - **反双曲正切：** $\text{artanh } x = \frac{1}{2}\ln\left(\frac{1+x}{1-x}\right)$

- **应用：** 在积分中，若标准三角换元过于繁琐，使用**双曲换元** ($x=a \sinh t$ 或 $x=a \cosh t$) 通常可以更直接地得到这些对数形式的结果。

## 3. 含 $x^2$ 的六种基本积分归纳

这六个公式是微积分中处理分母含二次项（无根号和有根号）积分的核心工具。

### A. 无根号项积分 (对数/反正切形式)

1. $\int \frac{1}{a^2-x^2} dx$
$$\frac{1}{2a} \ln\left|\frac{a+x}{a-x}\right| + C$$
    - **记忆点：** 常数 $a$ 领先（$a^2$ 是正的），因此在分子上写 $\mathbf{a}$ 领先的项 $\mathbf{a+x}$。

1. $\int \frac{1}{x^2-a^2} dx$
    
    $$\frac{1}{2a} \ln\left|\frac{x-a}{x+a}\right| + C$$

    - **记忆点：** 变量 $x$ 领先（$x^2$ 是正的），因此在分子上写 $\mathbf{x}$ 领先的项 $\mathbf{x-a}$。

2. $\int \frac{1}{a^2+x^2} dx$
$$\frac{1}{a} \arctan\left(\frac{x}{a}\right) + C$$
    - **记忆点：** 分母为 $\mathbf{a^2+x^2}$ 且**不带根号** $\implies$ **反正切** ($\arctan$)。

### B. 根号项积分 (反三角/反双曲形式)

1. $\int \frac{1}{\sqrt{a^2-x^2}} dx$
$$\arcsin\left(\frac{x}{a}\right) + C$$
    - **记忆点：** $\sqrt{\mathbf{a^2}-x^2}$（常数领先） $\implies$ **反正弦** ($\arcsin$)。
2. $\int \frac{1}{\sqrt{x^2+a^2}} dx$
$$\ln|x+\sqrt{x^2+a^2}| + C$$
    - **记忆点：** $\sqrt{\mathbf{x^2}+a^2}$（$\mathbf{+}$ 号） $\implies$ **反双曲正弦** $\text{arsinh}$ 形式。

3. $\int \frac{1}{\sqrt{x^2-a^2}} dx$
$$\ln|x+\sqrt{x^2-a^2}| + C$$
    - **记忆点：** $\sqrt{\mathbf{x^2}-a^2}$（$\mathbf{-}$ 号） $\implies$ **反双曲余弦** $\text{arcosh}$ 形式。
---

### 🔑 记忆关键点总结

- **系数规则：** 结果是 $\arctan$ 或 $\arcsin$ 的，前面系数是 $\mathbf{\frac{1}{a}}$。结果是 $\ln$ 形式的，前面系数是 $\mathbf{\frac{1}{2a}}$ 或 $\mathbf{1}$ (取决于是否有根号)。
    
- **三角 vs. 双曲：** 根号下 $\mathbf{a^2-x^2}$ 是**三角函数**；$\mathbf{x^2 \pm a^2}$ 是**双曲函数**。
