## 1. 次数差法则

针对 $\int R(\sin x, \cos x) dx$，观察分子分母的最高次数 $n_{up}$ 和 $n_{down}$：

|**次数关系**|**目标构造**|**必杀技**|**适用场景**|
|---|---|---|---|
|**差 1 次**|$d(\sin x)$ 或 $d(\cos x)$|**留项法**|分母为关于 $\sin x$ 或 $\cos x$ 的多项式|
|**差 2 次**|$d(\tan x)$|**弦化切 (同除 $\cos^2 x$)**|偶函数型，$f(-s, -c) = f(s, c)$|
|**次数相等**|$d(x \pm a)$|**拆项加减**|如 $\int \frac{1}{1+\tan x} dx$，构造 $\frac{(\sin+\cos)+(\cos-\sin)}{\sin+\cos}$|

## 2. 对称性与轮换对称

北大卷子经常出现涉及 $\sin x$ 与 $\cos x$ 对称的结构。

- **对称式 ($\sin^4 x + \cos^4 x$)**：优先利用二倍角公式 $2x$ 降维，再应用次数差法则。
    
- **轮换对称（定积分常考）**：利用区间再现公式 $\int_0^{\pi/2} f(\sin x) dx = \int_0^{\pi/2} f(\cos x) dx$，直接让分母消失。

## 3. $f(\tan x) dx$的代换

当你面对 $\int f(\tan x) dx$ 时，因为 $dx = \frac{1}{1+\tan^2 x} d(\tan x)$，原式其实恒等于：

$$\int \frac{f(u)}{1+u^2} du \quad (u = \tan x)$$

## 4. $\sec^2x$与$\tan x$搭配的算子平衡

对于某些特殊的 $f(\tan x)$，利用 $\tan x$ 的对称性或者分项式，可以构造出一种 **“分部积分的简写版”** 或者 **“算子平衡”**。

比如，北大期末常考的一类题型：$\int (\tan^n x + \tan^{n-2} x) dx$。

如果你顺着这个思路，你会发现：

$$\tan^n x + \tan^{n-2} x = \tan^{n-2} x (\tan^2 x + 1) = \tan^{n-2} x \cdot \sec^2 x$$

这直接凑出了 $d(\tan x)$，结果就是：

$$\int (\tan^n x + \tan^{n-2} x) dx = \frac{\tan^{n-1} x}{n-1} + C$$

这种 **“成对出现、阶数差2”** 的观察法，是处理高次 $\tan x$ 的绝招。
