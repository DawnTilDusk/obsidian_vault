
# 一
说实话，在选这门《现代工学通论》之前，我对新工科的理解其实挺传统的。在我的印象里，学机器人的可能就是整天跟电机、减速器死磕，或者在工厂里调那些笨重的机械臂。但是听完马晓光教授关于具身智能的报告，我当时在台下确实体验到了有一种豁然开朗的感觉。

马教授开始提到的那个核心概念——人工智能，比如ChatGPT和deepseek，他们正在从互联网的纯软件形态，通过物理身体进入真实的物理世界。这直接击中了我最感兴趣的点。编写软件让我感觉自己在进行创造，而软件控制硬件，则让我感受到了自己创造的现实价值。我开始意识到，我们正处在一个巨大的技术分水岭上。以前我们做的是自动化，是为了让机器重复人做过的事；而现在，我们要赋予机器身体和智能，让它们像人一样去感知、去决策。这种从被动的工具到主动的智能体的跨越，让我对机器人工程这个专业有了全新的认识。

以前我觉得机器人就是机械+电路，但马教授的那张关于低空系统发展历程的PPT，也就是讲座中关于无人机四代演变的那个图，让我印象特别深刻。从第一代还得靠人手忙脚乱遥控的航模，到第二代有了飞控能自己稳住，再到第四代具身智能无人机——这完全是两个物种了。现在的机器人工程，核心已经不仅仅是把硬件搭起来，而是怎么给这个铁壳子装上大脑和小脑。

说到这个，讲座里提到的“大脑-小脑”分层架构让我琢磨了很久。在这个体系里，机器人工程不再是一个单纯的硬件学科，而变成了一个超级交叉学科。我们需要用大模型来做逻辑推理，告诉机器人“我渴了”，它得能理解这意味着要去厨房拿杯子倒水；同时，还需要精密的运动控制算法来保证它走路不摔跤，手拿杯子时力度刚好。

所以我现在的理解是：未来的机器人工程，其实是物理学、控制论和人工智能的深度融合。我们要解决的不仅仅是让机器人怎么动的问题，更是让具身智能怎么理解我们人类需求的问题。这比单纯写代码或者单纯画图纸要难得多，但也感性得多， 让我非常着迷。

在报告中，我最感兴趣，也是听得最仔细的部分，是关于IR模仿学习那一块。
我在PPT里看到了斯坦福的ALOHA系统和各种机械臂操作的动图，看着机器人笨拙地尝试叠衣服、倒水，感觉既好笑又震撼。马老师提到现在的痛点是“端到端响应速度”和“分布外偏差”。但是其实我觉得通俗点说，就是机器人现在还太慢，延迟太大，而且稍微换个环境，比如杯子颜色变了，或者光线暗了，它可能就傻了。

这里面涉及到一个叫TSF-IL（中文翻译应该是融合时序和阶段信息的多模态模仿学习）的算法，我看到报告里ppt中展示的数据对比非常直观：传统的ACT算法成功率只有52%，而引入了阶段划分和时序融合后，成功率直接飙升到了76%。这让我意识到，在硬件标准逐渐统一的今天，算法才是那个决定胜负的关键。

另外，马老师课堂上为我们演示的LeRobot开源项目和低成本遥操作手柄的案例也让我很兴奋。以前觉得搞具身智能得有几百万的实验室设备，但现在通过开源社区和3D打印的低成本设备，甚至用个游戏手柄就能采集数据训练机器人。这种技术不在高大上的趋势，意味着像我这样的学生也有机会在宿舍或者学校实验室里，动手复现一些最前沿的具身智能Demo。这让我觉得这个看起来无比高大上的领域其实离我并不远。正如马老师在课堂上所提到的那样，趁着年轻，多复现一个项目，就多一份经验，自己的能力也在其中不断增长。
# 二
结合报告里面PPT里提到的市场现状——2030年市场规模要到2000多亿，而且算法工程师岗位增长了479%，薪资涨幅高达100%——说不动心是假的。但抛开钱的因素，我更看重的是这个方向的成长性。所以，我计划将具身智能算法工程师作为我的职业目标。具体的准备路径，我有这么几个不成熟的想法：

1. 死磕数学和理论基础：既然是搞算法，线性代数、概率论和凸优化是跑不掉的。以前觉得数学枯燥，现在知道它是为了让机器人不摔跤，动力就足多了。
2. 尝试Sim-to-Real：既然真机实验成本高，我打算先从仿真软件Isaac Gym入手。报告里也强调了仿真数据的训练价值，我希望能在本科生研究或者研究生阶段熟练掌握在虚拟环境中训练模型，然后迁移到真机的技术。
3. 关注数据采集技术：马老师提到数据是壁垒。以后我会特别关注怎么高效地收集高质量的机器人交互数据，比如学习如何使用遥操作设备，甚至尝试自己去开源社区贡献一些数据。
4. 多实践项目：我觉得作为一个工科学生，我应该学会在实践中去培养自己的工程能力，包括建模，接线，嵌入式开发，软件debug等等，这些课堂外的东西也是一个工程师的必备素养

我现在已经在参与一个机器狗自动避障导航的NAV_RL项目，我在其中已经感受到了很多具身智能的魅力所在。我希望未来自己能参与到一款真正能走进某个机械臂或者说是家庭的服务机器人的研发中，不求它能像科幻电影里那么神，但至少能帮家里老人稳稳地倒杯水，或者把乱糟糟的房间收拾干净。我觉得这不仅仅是技术，也是一种成就感，一种人文关怀。
# 三、 课程收获总结

这学期的《现代工学通论》给我的最大收获，与其说是知识点的堆砌，不如说是工程视野的扩张。以前我看待专业课，就像是在看一块块独立的拼图，高数是高数，力学是力学，代码是代码。但通过这几场讲座，尤其是马老师这场，我看到了这些拼图拼在一起的样子。我开始明白，我们去学习“新工科”，其实就是去学习面对复杂问题时，那种不被单一学科限制、调用一切工具去解决问题的能力。这种思维方式的转变，我觉得比记住几个公式要值钱得多。
# 四、 对课程的建议

最后，想对这门课提一点小小的建议：
现在的讲座形式虽然干货很多，但大部分时间还是我们坐在台下听。如果可能的话，能不能在未来的课程里多增加一些类似马老师课堂里面的demo展示的环节？那天体验了Lerobot的机械臂操作让我非常受益（虽然之后我在极客实验室与学长一起搭了一个几乎一样的lerobot主从臂没有跑通），我觉得带来的体验感会比单纯看PPT更强。
另外，我也希望能多邀请一些像马老师这样在产业界也有实战经验， 视野开阔的专家，因为他们的视角和格局开阔，而且比较注重产教结合。这其实不仅是让我们知道象牙塔外面的世界正在发生什么，也为我们之后的学习方向提出了一些很有价值的建议
