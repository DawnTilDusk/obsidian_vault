以下是Beteflight地面站 全部禁止解锁标志（Arming Disable Flags）  
中文解释 + 原因 + 解决方法 整理，欢迎大家过来讨论，路过点个赞哦！  
  
**一、连接/安全/系统类**  
  
MSP  
  
- 解释：已连接地面站（USB/网页），安全禁止解锁  
- 原因：连电脑/网页时防止意外启动电机  
- 解决：拔掉USB；或在“电机”页勾选“我已了解风险”  
  
BOOT_GRACE  
  
- 解释：上电延时保护中（默认5秒）  
- 原因：防止上电瞬间传感器不稳就解锁  
- 解决：等待5秒后再解锁  
  
LOAD  
  
- 解释：系统负载过高（CPU占用过高）  
- 原因：功能开太多、循环率过高  
- 解决：降低PID循环率、关闭多余功能（黑盒、OSD）  
  
REBOOT_REQUIRED  
  
- 解释：参数修改后需重启飞控  
- 原因：修改关键设置未重启  
- 解决：保存并重启飞控  
  
USB_STUCK  
  
- 解释：USB状态异常  
- 原因：USB插拔后状态未复位  
- 解决：重新上电（拔插电池）  
  
  
  
**二、接收机/遥控类**  
  
RX_FAILSAFE / RXLOSS  
  
- 解释：无接收机信号 / 失控保护  
- 原因：遥控关机、未对频、接线错误  
- 解决：遥控器开机、对频、检查接线与协议  
  
BAD_RX  
  
- 解释：接收机信号质量差  
- 原因：信号干扰、天线问题、距离过远  
- 解决：优化天线、远离干扰源、重新对频  
  
RX_SNR  
  
- 解释：接收机信噪比过低  
- 原因：信号质量太差（CRSF/ELRS）  
- 解决：改善天线位置、降低发射功率  
  
FAILSAFE_SW  
  
- 解释：失控保护开关已激活  
- 原因：遥控器失控开关处于开启位  
- 解决：将开关切回正常位置  
  
  
  
三、油门/电机/电调类  
  
THROTTLE  
  
- 解释：油门未在最低位  
- 原因：摇杆未归底、校准不准、微调过高  
- 解决：油门推到底；重新校准遥控器  
  
THROTTLE_HIGH  
  
- 解释：油门过高（超过解锁阈值）  
- 原因：解锁时油门值>1050  
- 解决：确认油门归零；检查通道反向  
  
MOTOR_PROTO  
  
- 解释：电机协议未就绪/未接电池  
- 原因：未接动力电池、协议选择错误  
- 解决：必须接电池；在端口页选对DShot协议  
  
DSHOT_TELEM  
  
- 解释：DShot遥测未就绪  
- 原因：未接电池、未开启双向DShot  
- 解决：接电池；电机页勾选“双向DShot”  
  
RPMFILTER  
  
- 解释：RPM滤波未就绪  
- 原因：依赖DShot遥测，未接电池  
- 解决：接电池后自动消失  
  
RUNAWAY_TAKEOFF  
  
- 解释：防暴冲保护触发  
- 原因：解锁后油门异常升高  
- 解决：重新上锁再解锁；检查电机是否堵转  
  
MOTOR_STUCK  
  
- 解释：电机堵转/过流  
- 原因：桨叶卡住、电机短路、轴承损坏  
- 解决：断电检查桨叶、电机、线路  
  
  
  
**四、传感器/校准类**  
  
CALIB  
  
- 解释：传感器正在校准中  
- 原因：陀螺仪/加速度计/罗盘校准未完成  
- 解决：飞机水平静止，等待校准结束  
  
ACC_CALIBRATION  
  
- 解释：加速度计校准仍在进行中  
- 原因：校准未完成、飞机晃动、平面不平  
- 解决：重新校准加速度计；完全静止等待完成  
  
NO_ACC_CAL  
  
- 解释：加速度计未校准  
- 原因：校准丢失、固件重置  
- 解决：重新校准加速度计  
  
NOGYRO  
  
- 解释：未检测到陀螺仪  
- 原因：陀螺仪硬件故障、通信失败  
- 解决：重启飞控；仍异常则硬件损坏  
  
IMU_TEMP  
  
- 解释：陀螺仪/加速度计温度异常  
- 原因：飞控过热或环境过冷  
- 解决：降温后使用；加强散热  
  
MAG_CALIBRATING  
  
- 解释：罗盘（磁力计）正在校准  
- 原因：罗盘校准过程中  
- 解决：按提示旋转飞机，完成后消失  
  
MAG_INVALID  
  
- 解释：罗盘数据无效/受干扰  
- 原因：靠近电机/电调、校准不良  
- 解决：重新校准罗盘；远离干扰源  
  
ANGLE  
  
- 解释：飞机倾斜过大（超过最大解锁角度）  
- 原因：未水平放置、飞控安装歪斜  
- 解决：水平放置飞机；重新校准加速度计  
  
  
  
**五、开关/模式/安全逻辑类**  
  
ARM_SWITCH  
  
- 解释：解锁开关处于不安全位置  
- 原因：上电时解锁开关已在解锁位  
- 解决：先切到上锁位，再尝试解锁  
  
NOPREARM  
  
- 解释：未激活预解锁（Prearm）  
- 原因：设置了Prearm+Arm双开关，但Prearm未开  
- 解决：先打开Prearm开关，再解锁  
  
KILLSWITCH  
  
- 解释：急停开关（Kill Switch）已激活  
- 原因：急停开关处于停止位  
- 解决：切回正常位置  
  
CRASH_FLIP  
  
- 解释：翻车翻转模式（反乌龟）激活  
- 原因：翻转开关打开  
- 解决：关闭翻转模式开关  
  
LAUNCH_CONTROL  
  
- 解释：弹射控制模式激活  
- 原因：弹射开关打开  
- 解决：关闭弹射开关  
  
SAFE_MODE  
  
- 解释：安全模式激活  
- 原因：低电压、失控等触发安全保护  
- 解决：恢复正常条件后重新解锁  
  
  
  
**六、GPS/导航类  
**  
GPS  
  
- 解释：GPS未定位（卫星数不足）  
- 原因：室内/遮挡、搜星中  
- 解决：到室外空旷处等待定位（≥8颗星）  
  
  
  
**七、OSD/黑盒/其他  
**  
CMS / MENU  
  
- 解释：OSD菜单已打开  
- 原因：遥控器呼出OSD未退出  
- 解决：退出OSD菜单；重新上电  
  
GFX / OSD  
  
- 解释：OSD初始化未完成  
- 原因：刚上电OSD未就绪  
- 解决：等待几秒；重新上电  
  
BLACKBOX  
  
- 解释：黑盒正在录制/忙  
- 原因：SD卡慢、已满、写入中  
- 解决：等待停止；更换高速卡；关闭黑盒  
  
OVR_CHECKS  
  
- 解释：过压/过流/过温/低压保护  
- 原因：电压过低、短路、温度过高  
- 解决：更换满电电池；检查短路；降温  
  
  
  
✅ 正常状态  
  
- 连网页版：只显示 MSP  
- 不连电脑：无任何标志 → 可正常解锁
