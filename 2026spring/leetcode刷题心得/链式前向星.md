
## 0. 如何理解两个数组
---
把链式前向星想象成一个**书架**：
- **`edge` 数组**：是书架上所有的**书**（边）。它们虽然排在一起，但内容（起点终点）是乱的。
- **`head` 数组**：是书架的**索引标签**（比如：5 号柜从哪本书开始看）。
- **`next` 变量**：是书里面的**书签**（告诉你下一本相关的书在哪）。

>[!IMPORTANT]
>**如果你只想要所有的书，索引确实没用；**
>**但如果你想找“关于点 A 的书”，没有索引你就会找疯掉。**

## 1. 核心：如何遍历某一点 $u$ 的所有出边？
---

这是链式前向星最常用的操作。我们通过 `head[u]` 找到最后加入的那条边，然后利用边结构体里的 `next` 属性像“拨念珠”一样往前跳。
```cpp
// 假设 head[u] 存储的是点 u 的第一条边下标
// edge[i].next 存储的是下一条边的下标
// 当 i 为 0 或 -1 时（取决于你的初始化），表示没有更多边了

for (int i = head[u]; i != 0; i = edge[i].next) {
    int v = edge[i].to;   // 终点
    int w = edge[i].w;    // 边权
    // 在这里对邻居节点 v 执行操作
}
```
## 2. 如何按顺序遍历“整个图”的所有点？
---
链式前向星本身只是存储结构，要遍历全图的所有点，你依然需要配合 **DFS（深度优先搜索）** 或 **BFS（广度优先搜索）**。

### 深度优先搜索 (DFS) 示例

这是最常见的全图遍历方式，常用于检查连通性或寻找路径。

```cpp
bool visited[MAXN];

void dfs(int u) {
    visited[u] = true;
    cout << "访问到了点：" << u << endl;

    // 关键：遍历当前点 u 的所有邻居
    for (int i = head[u]; i != 0; i = edge[i].next) {
        int v = edge[i].to;
        if (!visited[v]) {
            dfs(v);
        }
    }
}

// 在 main 函数中
for (int i = 1; i <= n; i++) {
    if (!visited[i]) dfs(i); // 处理非连通图
}
```
### 为什么这个 `for` 循环长得这么奇怪？

很多初学者会被 `i = edge[i].next` 绕晕。我们来拆解一下加边和遍历的过程：
1. **加边时**：新边总是插在链表的“头部”。
    - 新边的 `next` 指向当前的 `head[u]`。
    - `head[u]` 更新为这条新边的编号。
2. **遍历时**：
    - 我们从最新的边（`head[u]`）开始。
    - 通过 `next` 找到上一次加的边。
    - 直到 `next` 指向 0（初始值），说明到头了。

> [!小技巧]
> 在初始化时，习惯上把 `head` 数组全部置为 `0`。这样在 `for` 循环中，判定条件 `i != 0` 就像是链表里的 `ptr != NULL`。

## 3. 如何盲遍历图的每一条边
---
如果你需要频繁地“盲遍历”所有边且需要知道起点，建议在结构体里加一个 `from` 变量：

```cpp
struct Edge {
    int from, to, w, next; // 加上 from
} edge[MAXE];

/*或者写成：
struct Edge {
    int u, v, w, nxt;
} edge[MAXE];*/

// 只要遍历 1 到 cnt 即可覆盖图中所有的边
for (int i = 1; i <= cnt; i++) {
    int from = edge[i].from;
    int to = edge[i].to;
    int weight = edge[i].w;
    
    // 处理这条边...
}
```

```cpp
struct Edge {
    int to, next, w; // to: 终点, next: 同起点的下一条边, w: 权重
} edge[MAXE];

int head[MAXN], cnt;

void add_edge(int u, int v, int w) {
    edge[++cnt].to = v;
    edge[cnt].w = w;
    edge[cnt].next = head[u];
    head[u] = cnt;
}
```
