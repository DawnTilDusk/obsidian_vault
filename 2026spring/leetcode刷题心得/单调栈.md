### 1. 核心定义

**单调栈** 是一种特殊的栈结构。在推入（Push）新元素时，通过 **弹出（Pop）** 破坏单调性的元素，从而保证栈内元素始终保持 **单调递增** 或 **单调递减** 的顺序。

- **存储内容：** 通常存储数组的 **下标 (Index)**，而不是直接存储数值（存下标既能找到值，也能计算距离）。

---

### 2. 运作逻辑与类型

单调栈的核心在于“**及时清理**”。当新来的元素破坏了当前的单调规则时，就说明栈顶元素的“命运”已经确定了（找到了它的下一个更值），此时将其弹出并处理。
#### A. 单调递增栈 (Monotone Increasing Stack)
- **规则：** 栈内元素从栈底到栈顶单调递增（如 `[1, 3, 5]`）。
- **操作：** 当新元素 `current` **小于** 栈顶元素时，破坏了递增，需要不断弹出栈顶，直到栈顶小于等于 `current`。
- **功能：** 寻找左边/右边 **第一个比自己小** 的元素。

#### B. 单调递减栈 (Monotone Decreasing Stack)

- **规则：** 栈内元素从栈底到栈顶单调递减（如 `[5, 3, 1]`）。
- **操作：** 当新元素 `current` **大于** 栈顶元素时，破坏了递减，需要不断弹出栈顶，直到栈顶大于等于 `current`。
- **功能：** 寻找左边/右边 **第一个比自己大** 的元素。

---

### 3. 使用条件与适用场景

当你看到题目中包含以下特征时，首先想到单调栈：

1. **“下一个更大/更小元素” (Next Greater/Smaller Element)**
    - 题目需要你在一个数组中，为每个元素找到左边或右边第一个比它大（或小）的元素。
2. **“左右边界”问题**
    - 需要确定一个元素向左右两边能延伸多远（例如：柱状图中最大的矩形，某个高度能维持多宽）。
3. **区间最值相关的贡献度**
    - 例如：求所有子数组的最小值之和（需要知道每个元素作为最小值能覆盖多大的范围）。
4. **时间复杂度要求**
    - 暴力解法通常是 $O(N^2)$，而题目要求 $O(N)$。

### 4. 通用代码模板 (Python)

这是解决此类问题的通用骨架（以**找下一个更大元素**为例，使用单调递减栈）：
```python
def nextGreaterElements(nums):
    n = len(nums)
    # 存放答案的数组，初始化为 -1 或 0
    res = [-1] * n 
    # 初始化栈
    stack = [] 

    for i in range(n):
        # 【核心逻辑】
        # 当栈不为空，且当前元素 > 栈顶元素对应的值时
        # 说明我们找到了栈顶元素的 "Next Greater Element"
        while stack and nums[i] > nums[stack[-1]]:
            index = stack.pop()
            res[index] = nums[i]  # 记录答案
        # 将当前索引入栈，等待它的 "Next Greater Element" 出现
        stack.append(i)
        
    return res
```
```cpp
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        stack<pair<int, int>> st;
        vector<int> ans(temperatures.size());
        for(int i=0; i<temperatures.size(); i++){
            int cur = temperatures[i];
            while(!st.empty() && cur > st.top().first){
                ans[st.top().second] = i - st.top().second;
                st.pop();
            }
            st.push({temperatures[i], i});
        }
        return ans;
    }
};
```

### 5. 经典题目分类

为了掌握单调栈，建议按以下顺序刷题：

#### 第一阶段：基础应用 (Next Greater Element)

最直观的应用，直接套用模板。

- **LeetCode 739. 每日温度 (Daily Temperatures)** - _最经典入门题_
- **LeetCode 496. 下一个更大元素 I**
- **LeetCode 503. 下一个更大元素 II** (涉及循环数组，需遍历两次或取模)

#### 第二阶段：接雨水与矩形面积 (二维思维)

不仅仅是找大小，还需要利用下标计算宽度（Width = 当前下标 - 栈顶下标 - 1）。

- **LeetCode 42. 接雨水 (Trapping Rain Water)** - _面试高频，单调递减栈_
- **LeetCode 84. 柱状图中最大的矩形 (Largest Rectangle in Histogram)** - _单调递增栈，找左右边界_

#### 第三阶段：进阶与贡献法 (Contribution Technique)

需要计算每个元素在多少个子数组中充当最大/最小值。

- **LeetCode 907. 子数组的最小值之和 (Sum of Subarray Minimums)**
- **LeetCode 85. 最大矩形 (Maximal Rectangle)** - _基于 LeetCode 84 的变体_

### 6. 总结速查表

|**目标 (Find)**|**栈的类型**|**操作逻辑 (While Condition)**|
|---|---|---|
|**右边第一个更大**|单调递减栈|`curr > stack_top`|
|**右边第一个更小**|单调递增栈|`curr < stack_top`|
|**左边第一个更大**|单调递减栈|`curr > stack_top` (弹出后，新的 `stack_top` 即为左边第一个大)|
|**左边第一个更小**|单调递增栈|`curr < stack_top` (弹出后，新的 `stack_top` 即为左边第一个小)|
