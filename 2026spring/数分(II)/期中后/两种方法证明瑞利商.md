瑞利商（Rayleigh Quotient）的最值问题是线性代数中极其优美且重要的一个结论。它的核心结论是：**对于实对称矩阵，其瑞利商的最大值等于矩阵的最大特征值，最小值等于矩阵的最小特征值。**

为了让你全面理解，我将为你提供两种最经典的证明方法：**代数法**（利用特征值分解，最严谨直观）和**微积分法**（利用拉格朗日乘数法，最具几何意义）。

在开始证明之前，我们先明确瑞利商的定义。设 $A$ 是一个 $n \times n$ 的实对称矩阵，$x$ 是一个非零的 $n$ 维列向量，瑞利商定义为：

$$R(x) = \frac{x^T A x}{x^T x}$$

---

### 方法一：代数法（基于正交对角化）

这个证明的核心思想是“坐标系变换”，把原本耦合在一起的二次型，变换到一个新的正交基底上，使其彻底解耦。

#### **1. 引入谱定理（Spectral Theorem）**

因为 $A$ 是实对称矩阵，根据实对称矩阵的性质，它一定可以被正交对角化。即存在一个正交矩阵 $Q$（满足 $Q^T Q = I$）和一个对角矩阵 $\Lambda$，使得：

$$A = Q \Lambda Q^T$$

其中，对角矩阵 $\Lambda$ 的对角线元素就是 $A$ 的 $n$ 个实特征值。我们不妨将它们按从大到小的顺序排列：

$$\lambda_1 \ge \lambda_2 \ge \dots \ge \lambda_n$$

这里的 $\lambda_1$ 就是最大特征值 $\lambda_{\max}$，$\lambda_n$ 就是最小特征值 $\lambda_{\min}$。

#### **2. 变量代换**

令 $x = Q y$（或者等价地写成 $y = Q^T x$）。因为 $Q$ 是正交矩阵，这种变换相当于对向量进行了旋转，但不改变其长度。

我们分别计算瑞利商的分子和分母：

- **分母：** $x^T x = (Qy)^T (Qy) = y^T Q^T Q y = y^T I y = y^T y = \sum_{i=1}^n y_i^2$
    
- **分子：** $x^T A x = x^T (Q \Lambda Q^T) x = (Q^T x)^T \Lambda (Q^T x) = y^T \Lambda y = \sum_{i=1}^n \lambda_i y_i^2$
    

#### **3. 转化为标量不等式**

将代换后的分子和分母代回瑞利商：

$$R(x) = \frac{\sum_{i=1}^n \lambda_i y_i^2}{\sum_{i=1}^n y_i^2}$$

#### **4. 证明最大值**

因为对于所有的 $i$，都有 $\lambda_i \le \lambda_1$，所以我们可以对分子进行放缩：

$$\sum_{i=1}^n \lambda_i y_i^2 \le \sum_{i=1}^n \lambda_1 y_i^2 = \lambda_1 \sum_{i=1}^n y_i^2$$

将其代入瑞利商表达式：

$$R(x) \le \frac{\lambda_1 \sum_{i=1}^n y_i^2}{\sum_{i=1}^n y_i^2} = \lambda_1$$

**取等号条件：** 当且仅当 $y = (1, 0, \dots, 0)^T$ 时等号成立。反推回原坐标系，这对应于 $x = Qy = q_1$（即 $\lambda_1$ 对应的特征向量）。

#### **5. 证明最小值**

同理，因为对于所有的 $i$，都有 $\lambda_i \ge \lambda_n$：

$$\sum_{i=1}^n \lambda_i y_i^2 \ge \sum_{i=1}^n \lambda_n y_i^2 = \lambda_n \sum_{i=1}^n y_i^2$$

代入瑞利商表达式得：

$$R(x) \ge \frac{\lambda_n \sum_{i=1}^n y_i^2}{\sum_{i=1}^n y_i^2} = \lambda_n$$

**取等号条件：** 当且仅当 $y = (0, \dots, 0, 1)^T$ 时等号成立，对应于 $x$ 为 $\lambda_n$ 对应的特征向量 $q_n$。

**结论：** $\lambda_{\min} \le R(x) \le \lambda_{\max}$ 证明完毕。

---

### 方法二：微积分法（拉格朗日乘数法）

这个方法的物理直觉更强。注意观察瑞利商 $R(x)$，如果我们将 $x$ 乘以一个常数 $k \neq 0$，会发现 $R(kx) = \frac{k^2 x^T A x}{k^2 x^T x} = R(x)$。

也就是说，**瑞利商的值与向量的长度无关，只跟向量的方向有关。**

因此，为了求最值，我们完全可以人为地加上一个长度约束：限制 $x$ 在单位球面上，即要求 $x^T x = 1$。

此时，求 $R(x)$ 的最值问题，就等价于：

**在条件 $x^T x = 1$ 的约束下，求 $f(x) = x^T A x$ 的条件极值。**

#### **1. 构造拉格朗日函数**

引入拉格朗日乘子 $\lambda$，构造函数：

$$L(x, \lambda) = x^T A x - \lambda (x^T x - 1)$$

#### **2. 求解偏导数寻找驻点**

对向量 $x$ 求梯度（利用矩阵求导法则 $\nabla_x (x^T A x) = 2Ax$，$\nabla_x (x^T x) = 2x$）：

$$\nabla_x L = 2Ax - 2\lambda x$$

令梯度等于零求极值点：

$$2Ax - 2\lambda x = 0 \implies Ax = \lambda x$$

#### **3. 得出结论**

看！推导出来的结果 $Ax = \lambda x$ 正是矩阵特征值和特征向量的定义式。

这意味着：**目标函数在单位球面上的极值点（驻点），必定是矩阵 $A$ 的特征向量。**

我们将这个驻点条件 $Ax = \lambda x$ 代回原目标函数分子中：

$$f(x) = x^T A x = x^T (\lambda x) = \lambda (x^T x) = \lambda \cdot 1 = \lambda$$

这说明，在每一个极值点处，瑞利商的值就等于对应的特征值。显然，这些极值中的最大值就是最大特征值 $\lambda_{\max}$，最小值就是最小特征值 $\lambda_{\min}$。
