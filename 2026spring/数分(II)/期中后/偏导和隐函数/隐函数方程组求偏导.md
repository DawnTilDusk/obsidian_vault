## 例题

这是一道关于隐函数方程组求偏导的经典高等数学题目。我们需要分别对两种不同自变量选取的情况进行求解。

已知方程组：

$$\begin{cases} u + 2v + 3x + 4y + 5z = 3 \\ u + v^2 + x^3 + y^4 + z^5 = 0 \end{cases}$$

---
### 第 (1) 题

**已知：** 以 $x, y, z$ 为自变量，$u, v$ 为因变量。

**求解：** $\frac{\partial u}{\partial x}, \frac{\partial^2 u}{\partial x^2}, \frac{\partial^2 u}{\partial x \partial y}$

**1. 求一阶偏导数 $\frac{\partial u}{\partial x}$ 和 $\frac{\partial v}{\partial x}$**

将方程组两边同时对 $x$ 求偏导（此时将 $y, z$ 视为常数）：

$$\begin{cases} \frac{\partial u}{\partial x} + 2\frac{\partial v}{\partial x} + 3 = 0 \quad \text{①} \\ \frac{\partial u}{\partial x} + 2v\frac{\partial v}{\partial x} + 3x^2 = 0 \quad \text{②} \end{cases}$$

由①式可得 $2\frac{\partial v}{\partial x} = -3 - \frac{\partial u}{\partial x}$，代入②式中：

$$\frac{\partial u}{\partial x} + v \left(-3 - \frac{\partial u}{\partial x}\right) + 3x^2 = 0$$

$$(1 - v)\frac{\partial u}{\partial x} - 3v + 3x^2 = 0$$

解得：

$$\frac{\partial u}{\partial x} = \frac{3(v - x^2)}{1 - v}$$

同时，我们顺便解出 $\frac{\partial v}{\partial x}$ 备用：

$$2\frac{\partial v}{\partial x} = -3 - \frac{3v - 3x^2}{1 - v} = \frac{-3(1-v) - 3v + 3x^2}{1-v} = \frac{3x^2 - 3}{1 - v}$$

$$\frac{\partial v}{\partial x} = \frac{3(x^2 - 1)}{2(1 - v)}$$

**2. 求二阶偏导数 $\frac{\partial^2 u}{\partial x^2}$**

对 $\frac{\partial u}{\partial x}$ 继续关于 $x$ 求偏导（注意 $v$ 也是 $x$ 的函数）：

$$\begin{aligned} \frac{\partial^2 u}{\partial x^2} &= \frac{\partial}{\partial x} \left( \frac{3v - 3x^2}{1 - v} \right) \\ &= \frac{\left(3\frac{\partial v}{\partial x} - 6x\right)(1 - v) - (3v - 3x^2)\left(-\frac{\partial v}{\partial x}\right)}{(1 - v)^2} \\ &= \frac{3(1 - v)\frac{\partial v}{\partial x} - 6x(1 - v) + 3(v - x^2)\frac{\partial v}{\partial x}}{(1 - v)^2} \\ &= \frac{3(1 - x^2)\frac{\partial v}{\partial x} - 6x(1 - v)}{(1 - v)^2} \end{aligned}$$

将前面求得的 $\frac{\partial v}{\partial x} = \frac{3(x^2 - 1)}{2(1 - v)} = -\frac{3(1 - x^2)}{2(1 - v)}$ 代入上式：

$$\begin{aligned} \frac{\partial^2 u}{\partial x^2} &= \frac{3(1 - x^2) \left[ -\frac{3(1 - x^2)}{2(1 - v)} \right] - 6x(1 - v)}{(1 - v)^2} \\ &= \frac{-\frac{9(1 - x^2)^2}{2(1 - v)} - 6x(1 - v)}{(1 - v)^2} \\ &= -\frac{9(x^2 - 1)^2 + 12x(1 - v)^2}{2(1 - v)^3} \end{aligned}$$

**3. 求二阶混合偏导数 $\frac{\partial^2 u}{\partial x \partial y}$**

首先需要求出 $\frac{\partial v}{\partial y}$。将原方程组两边对 $y$ 求偏导：

$$\begin{cases} \frac{\partial u}{\partial y} + 2\frac{\partial v}{\partial y} + 4 = 0 \\ \frac{\partial u}{\partial y} + 2v\frac{\partial v}{\partial y} + 4y^3 = 0 \end{cases}$$

两式相减可消去 $\frac{\partial u}{\partial y}$：

$$(2 - 2v)\frac{\partial v}{\partial y} + 4 - 4y^3 = 0 \implies \frac{\partial v}{\partial y} = \frac{2(y^3 - 1)}{1 - v}$$

然后，对 $\frac{\partial u}{\partial x} = \frac{3v - 3x^2}{1 - v}$ 关于 $y$ 求偏导：

$$\begin{aligned} \frac{\partial^2 u}{\partial x \partial y} &= \frac{\partial}{\partial y} \left( \frac{3v - 3x^2}{1 - v} \right) \\ &= \frac{3\frac{\partial v}{\partial y}(1 - v) - (3v - 3x^2)\left(-\frac{\partial v}{\partial y}\right)}{(1 - v)^2} \\ &= \frac{3(1 - x^2)\frac{\partial v}{\partial y}}{(1 - v)^2} \end{aligned}$$

将 $\frac{\partial v}{\partial y}$ 代入：

$$\frac{\partial^2 u}{\partial x \partial y} = \frac{3(1 - x^2) \cdot \frac{2(y^3 - 1)}{1 - v}}{(1 - v)^2} = \frac{6(1 - x^2)(y^3 - 1)}{(1 - v)^3}$$

---

### 第 (2) 题

**已知：** 以 $x, y, v$ 为自变量，$u, z$ 为因变量。

**求解：** $\frac{\partial u}{\partial x}, \frac{\partial^2 u}{\partial x^2}, \frac{\partial^2 u}{\partial x \partial y}$

**1. 求一阶偏导数 $\frac{\partial u}{\partial x}$ 和 $\frac{\partial z}{\partial x}$**

将原方程组两边同时对 $x$ 求偏导（此时将 $y, v$ 视为常数）：

$$\begin{cases} \frac{\partial u}{\partial x} + 3 + 5\frac{\partial z}{\partial x} = 0 \quad \text{③} \\ \frac{\partial u}{\partial x} + 3x^2 + 5z^4\frac{\partial z}{\partial x} = 0 \quad \text{④} \end{cases}$$

由③式得 $5\frac{\partial z}{\partial x} = -3 - \frac{\partial u}{\partial x}$，代入④式：

$$\frac{\partial u}{\partial x} + 3x^2 + z^4\left(-3 - \frac{\partial u}{\partial x}\right) = 0$$

$$(1 - z^4)\frac{\partial u}{\partial x} + 3x^2 - 3z^4 = 0$$

解得：

$$\frac{\partial u}{\partial x} = \frac{3(z^4 - x^2)}{1 - z^4}$$

同时解出 $\frac{\partial z}{\partial x}$ 备用：

$$5\frac{\partial z}{\partial x} = -3 - \frac{3z^4 - 3x^2}{1 - z^4} = \frac{-3 + 3z^4 - 3z^4 + 3x^2}{1 - z^4} = \frac{3(x^2 - 1)}{1 - z^4}$$

$$\frac{\partial z}{\partial x} = \frac{3(x^2 - 1)}{5(1 - z^4)}$$

**2. 求二阶偏导数 $\frac{\partial^2 u}{\partial x^2}$**

对 $\frac{\partial u}{\partial x}$ 继续关于 $x$ 求偏导（注意 $z$ 也是 $x$ 的函数）：

$$\begin{aligned} \frac{\partial^2 u}{\partial x^2} &= \frac{\partial}{\partial x} \left( \frac{3z^4 - 3x^2}{1 - z^4} \right) \\ &= \frac{\left(12z^3\frac{\partial z}{\partial x} - 6x\right)(1 - z^4) - (3z^4 - 3x^2)\left(-4z^3\frac{\partial z}{\partial x}\right)}{(1 - z^4)^2} \\ &= \frac{12z^3(1 - x^2)\frac{\partial z}{\partial x} - 6x(1 - z^4)}{(1 - z^4)^2} \end{aligned}$$

将 $\frac{\partial z}{\partial x} = \frac{3(x^2 - 1)}{5(1 - z^4)} = -\frac{3(1 - x^2)}{5(1 - z^4)}$ 代入：

$$\begin{aligned} \frac{\partial^2 u}{\partial x^2} &= \frac{12z^3(1 - x^2) \left[ -\frac{3(1 - x^2)}{5(1 - z^4)} \right] - 6x(1 - z^4)}{(1 - z^4)^2} \\ &= \frac{-\frac{36z^3(1 - x^2)^2}{5(1 - z^4)} - 6x(1 - z^4)}{(1 - z^4)^2} \\ &= -\frac{36z^3(x^2 - 1)^2 + 30x(1 - z^4)^2}{5(1 - z^4)^3} \end{aligned}$$

**3. 求二阶混合偏导数 $\frac{\partial^2 u}{\partial x \partial y}$**

先求出 $\frac{\partial z}{\partial y}$。将原方程组两边对 $y$ 求偏导（$x, v$ 视作常数）：

$$\begin{cases} \frac{\partial u}{\partial y} + 4 + 5\frac{\partial z}{\partial y} = 0 \\ \frac{\partial u}{\partial y} + 4y^3 + 5z^4\frac{\partial z}{\partial y} = 0 \end{cases}$$

两式相减消去 $\frac{\partial u}{\partial y}$：

$$4 - 4y^3 + 5(1 - z^4)\frac{\partial z}{\partial y} = 0 \implies \frac{\partial z}{\partial y} = \frac{4(y^3 - 1)}{5(1 - z^4)}$$

然后，对 $\frac{\partial u}{\partial x} = \frac{3z^4 - 3x^2}{1 - z^4}$ 关于 $y$ 求偏导：

$$\begin{aligned} \frac{\partial^2 u}{\partial x \partial y} &= \frac{\partial}{\partial y} \left( \frac{3z^4 - 3x^2}{1 - z^4} \right) \\ &= \frac{12z^3\frac{\partial z}{\partial y}(1 - z^4) - (3z^4 - 3x^2)\left(-4z^3\frac{\partial z}{\partial y}\right)}{(1 - z^4)^2} \\ &= \frac{12z^3(1 - x^2)\frac{\partial z}{\partial y}}{(1 - z^4)^2} \end{aligned}$$

将 $\frac{\partial z}{\partial y}$ 的结果代入：

$$\frac{\partial^2 u}{\partial x \partial y} = \frac{12z^3(1 - x^2) \cdot \frac{4(y^3 - 1)}{5(1 - z^4)}}{(1 - z^4)^2} = \frac{48z^3(1 - x^2)(y^3 - 1)}{5(1 - z^4)^3}$$

## 对比一下上课所用的全微分待定系数法

从解题习惯和数学直觉来看，我们可以把这两种方法归纳为“精准打击”与“全局统筹”的区别。

### 1. 直接求偏导（公式法 / 链式法则）—— “精准打击”

这种方法的核心逻辑是：**缺什么，求什么，视无关变量为常数。**

- **思维习惯：** 这种思路非常像算法设计中的**贪心策略（Greedy Algorithm）**。你的目标很明确（比如只求 $\frac{\partial u}{\partial x}$），那么你就只针对 $x$ 这一条路径进行操作，将其余的 $y, z$ 强制冻结。
    
- **绝对优势：求解高阶导数。** 当题目要求二阶甚至更高阶的偏导数（如上一题中的 $\frac{\partial^2 u}{\partial x^2}$）时，直接对一阶导数的结果再次求导是最清晰、最不容易出错的。
    
- **局限性：** 视野较窄。如果题目要求你把所有的偏导数（$\frac{\partial u}{\partial x}, \frac{\partial u}{\partial y}, \frac{\partial v}{\partial x}, \frac{\partial v}{\partial y}$ 等等）全求一遍，你需要反复重复上述过程，步骤会变得繁琐。
    

### 2. 全微分法 —— “全局统筹”

这种方法的核心逻辑是：**保留所有变量的微小变化，建立完整的微分线性方程组。**

#### ==核心公式==
$$d^nu = (\vec{dx} · \nabla)^n u$$
$$\text{i.e. }d^nu = \bigg(\sum_{i= 1}^{m}\frac{\partial}{\partial x_i}dx_i\bigg)^nu$$

- **思维习惯：** 这种思路更具**系统性与代数美感**。它本质上是在提取整个方程组的雅可比矩阵（Jacobian Matrix）。在处理复杂的力学、相对论或者热力学系统推导时，由于常常需要考察多个物理量之间的微小耦合变化，$du, dv, dx$ 具有极其直观的物理意义，这种“一锅端”的方法非常契合分析物理系统的习惯。同时，从编程的角度看，全微分法高度格式化，非常容易转化为代码中的矩阵运算。
    
- **绝对优势：一次性求出所有一阶偏导数。** 你只需要将全微分方程组写出来，把因变量的微分（如 $du, dv$）用自变量的微分（如 $dx, dy, dz$）线性表出。方程整理好后，各个微分项前面的系数就直接对应了所有的偏导数。
    
- **局限性：** 求二阶导数极度痛苦。对一阶微分再次求微分会产生 $d^2u, dxdy$ 等极其复杂的二阶微分项，如果没有受过专门的张量或微分几何训练，很容易在符号海洋中迷失并算错。


---

### 归纳对比

为了更直观地对比，我们可以归纳出以下决策矩阵：

|**维度**|**直接求偏导法**|**全微分法**|
|---|---|---|
|**数学本质**|局部路径的链式法则应用|全局系统的线性化与矩阵求逆|
|**求解目标**|针对单一的、特定的偏导数|需要获取所有变量的一阶偏导关系|
|**计算高阶导数**|**极力推荐**（思路清晰，不易错）|**极不推荐**（二阶微分极易算错）|
|**方程组规模**|适合方程较少的情况|适合方程多、变量多的复杂系统|
|**思维定势**|“面向过程”：一步步顺藤摸瓜|“面向状态”：建立关系，提取系数|
