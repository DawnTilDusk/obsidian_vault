对于这类**参数方程形式**的曲面求切平面，我们通常有两套打法：“看破红尘法”（消元化隐）和“正面硬刚法”（叉乘求法向量）。我们把这两招都过一遍，你会发现微积分的底层逻辑极其统一。

### 方法一：“看破红尘法”（消元转化为隐函数）

当我们看到三角函数参数方程时，第一反应永远是**利用三角恒等式消去参数**。

仔细观察题目给定的方程组：

1. $\frac{x}{a} = \cos\theta \sec\varphi$
    
2. $\frac{y}{b} = \sin\theta \sec\varphi$
    
3. $\frac{z}{c} = \tan\varphi$
    

将前两式平方后相加，利用 $\cos^2\theta + \sin^2\theta = 1$ 消去 $\theta$：

$$\left(\frac{x}{a}\right)^2 + \left(\frac{y}{b}\right)^2 = (\cos^2\theta + \sin^2\theta)\sec^2\varphi = \sec^2\varphi$$

再联合第三式，利用三角恒等式 $\sec^2\varphi - \tan^2\varphi = 1$ 消去 $\varphi$：

$$\left(\frac{x}{a}\right)^2 + \left(\frac{y}{b}\right)^2 - \left(\frac{z}{c}\right)^2 = \sec^2\varphi - \tan^2\varphi = 1$$

你看！原形毕露了，它就是 $\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$。

根据我们在上一题推导出的结论或“劈开法”口诀，假设切点对应的参数为 $(\theta_0, \varphi_0)$，即切点坐标为 $(x_0, y_0, z_0)$，切平面方程直接秒写：

$$\frac{x_0 x}{a^2} + \frac{y_0 y}{b^2} - \frac{z_0 z}{c^2} = 1$$

把 $x_0, y_0, z_0$ 用参数替换回去，最终答案就是：

$$\frac{\cos\theta_0 \sec\varphi_0}{a}x + \frac{\sin\theta_0 \sec\varphi_0}{b}y - \frac{\tan\varphi_0}{c}z = 1$$

### 方法二：“正面硬刚法”（通用参数曲面解法）

并不是所有参数方程都能这么幸运地被消元。如果遇到无法消元的参数曲面 $\vec{r}(\theta, \varphi) = (x(\theta,\varphi), y(\theta,\varphi), z(\theta,\varphi))$，我们必须动用**通用武器：切向量叉乘**。

曲面上一点的两个偏导数向量 $\vec{r}_\theta$ 和 $\vec{r}_\varphi$ 是该点处的两个切向量，它们的叉积（Cross Product）就是我们梦寐以求的法向量 $\vec{n}$。

**1. 求两个切向量（对参数分别求偏导）**

$$\vec{r}_\theta = \left( \frac{\partial x}{\partial \theta}, \frac{\partial y}{\partial \theta}, \frac{\partial z}{\partial \theta} \right) = (-a\sin\theta\sec\varphi, \ b\cos\theta\sec\varphi, \ 0)$$

$$\vec{r}_\varphi = \left( \frac{\partial x}{\partial \varphi}, \frac{\partial y}{\partial \varphi}, \frac{\partial z}{\partial \varphi} \right) = (a\cos\theta\sec\varphi\tan\varphi, \ b\sin\theta\sec\varphi\tan\varphi, \ c\sec^2\varphi)$$

**2. 叉乘求法向量 $\vec{n} = \vec{r}_\theta \times \vec{r}_\varphi$**

经过行列式展开计算（这里跳过中间机械的代数运算），我们会得到一个极其庞大的向量：

$$\vec{n} = (bc\cos\theta\sec^3\varphi, \ ac\sin\theta\sec^3\varphi, \ -ab\sec^2\varphi\tan\varphi)$$

**3. 见证奇迹的时刻（化简法向量）**

法向量只代表方向，长度不重要。为了简化，我们把上面这个向量的所有分量同时乘以 $\frac{1}{abc \sec^2\varphi}$ 进行缩放，得到新的法向量 $\vec{n}'$：

$$\vec{n}' = \left( \frac{\cos\theta\sec\varphi}{a}, \frac{\sin\theta\sec\varphi}{b}, -\frac{\tan\varphi}{c} \right)$$

你仔细端详一下这个简化后的法向量的三个分量，结合题目给的参数方程，有没有觉得非常眼熟？

- $\frac{\cos\theta\sec\varphi}{a} = \frac{1}{a^2} (a\cos\theta\sec\varphi) = \frac{x}{a^2}$
- $\frac{\sin\theta\sec\varphi}{b} = \frac{1}{b^2} (b\sin\theta\sec\varphi) = \frac{y}{b^2}$
- $-\frac{\tan\varphi}{c} = -\frac{1}{c^2} (c\tan\varphi) = -\frac{z}{c^2}$

没错！我们用参数叉乘这种最暴力的几何方法算出来的法向量 $\vec{n}' = \left( \frac{x}{a^2}, \frac{y}{b^2}, -\frac{z}{c^2} \right)$，**与我们在上一题中对隐函数求梯度 $\nabla F$ 算出来的法向量，分毫不差地完全重合了！**

## 我们可以用经纬线的方式来理解法向量的求法

我们可以通过“编织曲面”的视角，分三步将这个过程在几何上彻底拆解：

### 第一步：理解参数方程 $\vec{r}(u, v)$ 在“编织”什么？

当一个曲面由参数方程 $\vec{r}(u, v) = \big(x(u,v), y(u,v), z(u,v)\big)$ 定义时，你可以把参数 $u$ 和 $v$ 想象成织布机上的经线和纬线。

- 如果我们**固定变量 $v$**（比如令 $v = v_0$），只让 $u$ 变动，那么空间中点 $\vec{r}(u, v_0)$ 的轨迹就会在曲面上画出一条曲线。这就好比曲面上的一条“经线”。
    
- 同理，如果我们**固定变量 $u$**（令 $u = u_0$），只让 $v$ 变动，点 $\vec{r}(u_0, v)$ 的轨迹会在曲面上画出另一条穿过它的曲线。这就是一条“纬线”。
    

曲面就是由无数条这样的“经线”和“纬线”交织而成的网。

### 第二步：偏导数的几何真面目——“经纬线”的切向量

在单变量微积分中，函数求导得到的是曲线的切线。在多变量的参数曲面中也是一样的：

- **对 $u$ 求偏导 $\vec{r}_u$：** 因为求偏导时 $v$ 被视为常数，所以 $\vec{r}_u = \frac{\partial \vec{r}}{\partial u}$ 在几何上代表的正是**沿着“经线”方向的切向量**。
    
- **对 $v$ 求偏导 $\vec{r}_v$：** 同理，$\vec{r}_v = \frac{\partial \vec{r}}{\partial v}$ 代表的是**沿着“纬线”方向的切向量**。
    

**关键点来了：** 因为“经线”和“纬线”都是紧紧贴在曲面上的曲线，所以它们的切向量 $\vec{r}_u$ 和 $\vec{r}_v$ 也必定紧紧贴在曲面上。换句话说，**这两个向量都躺在曲面该点的“切平面”内**。

### 第三步：叉乘的绝杀——从平面走向法线

现在，我们在曲面上某一点找到了两个不共线的切向量 $\vec{r}_u$ 和 $\vec{r}_v$。在几何上，**两个相交的直线（向量）可以唯一确定一个平面**。由 $\vec{r}_u$ 和 $\vec{r}_v$ 撑开的这个平面，就是曲面在该点的**切平面**。

接下来我们要找的是**法向量**（垂直于曲面/切平面的向量）。

在线性代数中，给你平面上的两个向量，怎么求垂直于它们的向量？**直接做叉乘！**

根据叉积（外积）的定义，向量 $\vec{a} \times \vec{b}$ 产生的新向量，必定同时垂直于 $\vec{a}$ 和 $\vec{b}$。

因此：

$$\vec{n} = \vec{r}_u \times \vec{r}_v$$

这个新生成的向量 $\vec{n}$，既垂直于 $u$ 方向的切线，又垂直于 $v$ 方向的切线。既然它垂直于切平面上的两个基础方向，那它自然就垂直于整个切平面，它就是我们要找的**法向量**！
