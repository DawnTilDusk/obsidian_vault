开映射定理：连续映射 开集的原像是开集
	但是开集不一定映射为开集
同胚映射
对于 $$ f : \mathbb{U} \rightarrow \mathbb{V}$$
f一一对应，且f连续，$f^{-1}$ 连续
则开集映射到开集，闭集映射到闭集


A的闭包为 包含A的所有闭集的交集

p, q, r, s 为只有两个元独立的一组变量
p, q为独立变量时：

$$\partial_p r - q\partial_q s + s = 0  \text{ and } \partial_p r \not= 0$$

证明：当p, r 为独立变量时:
$$\partial_p q - q\partial_r s + s\partial_r q = 0$$

全微分：
$$ds = \frac{\partial{s}}{\partial{p}}\Bigg|_q dp + \frac{\partial{s}}{\partial{q}}\Bigg|_p dq$$
$$dr = \frac{\partial{r}}{\partial{p}}\Bigg|_q dp + \frac{\partial{r}}{\partial{q}}\Bigg|_p dq$$
$$dq = \frac{\partial{q}}{\partial{p}}\Bigg|_r dp + \frac{\partial{q}}{\partial{r}}\Bigg|_p dr$$
$$\text{q 不变} \iff dr = \frac{\partial{r}}{\partial{p}}\Bigg|_q dp \iff dq = \frac{\partial{q}}{\partial{p}}\Bigg|_r dp + \frac{\partial{q}}{\partial{r}}\Bigg|_p dr = 0$$
$$\iff \frac{\partial{r}}{\partial{p}}\Bigg|_q = -\frac{\frac{\partial{q}}{\partial{p}}\bigg|_r}{\frac{\partial{q}}{\partial{r}}\bigg|_p}$$
