### 一、 Internet 基础架构与工作模式

#### 1. 基本概念

Internet 是在 **TCP/IP 协议**基础上建立的国际互联网。其核心要素包括：局域网/服务器、传输系统以及统一的 TCP/IP 协议栈。

#### 2. 三种主要工作模式

- **C/S 模式 (Client/Server)：** 客户端/服务器结构。任务分配在客户机和服务器之间。
- **B/S 模式 (Browser/Server)：** 浏览器/服务器结构。一种特殊的 C/S 模式，用户通过浏览器经由 HTTP 协议访问 Web 服务器，简化了客户端维护。
- **P2P 模式 (Peer-to-Peer)：** 对等模式。节点之间直接交换信息，不依赖中心服务器，每个节点既是资源提供者也是消费者。

---

### 二、 Internet 地址结构 (核心重点)

#### 1. IP 地址

- **作用：** 标识 Internet 上每台主机或路由器的逻辑地址。
- **IPv4 结构：** 32位二进制，通常用“点分十进制”表示（如 192.168.1.1）。
- **分类制 (Classful)：**
    - **A类：** 首位为0，网络号8位，适用于超大型网络。
    - **B类：** 前两位为10，网络号16位，适用于中型网络。
    - **C类：** 前三位为110，网络号24位，适用于小型局域网。
    - **D类：** 组播地址；
    - **E类：** 备用地址。

#### 2. 域名系统 (DNS)

- **功能：** 将便于人类记忆的域名（如 `www.pku.edu.cn`）转换为机器识别的 IP 地址。
- **层次结构：** 自右向左级别降低（顶级域名 -> 二级域名 -> 主机名）。

#### 3. 统一资源定位器 (URL)

- **格式：** `协议://主机名[:端口号]/路径/文件名`
- **常见协议：** http, https, ftp 等。

---

### 三、 Internet 接入技术

根据传输介质和技术不同，常见的接入方式包括：

1. **ADSL (非对称数字用户线路)：** 利用电话线传输，下行速率远高于上行。
    
2. **光纤接入 (FTTx)：** 目前最主流的高速接入方式，包括光纤到楼 (FTTB) 或光纤到户 (FTTH)。
    
3. **无线接入：** 包括 WiFi (WLAN) 和 4G/5G 移动网络接入。
    

---

### 四、 核心应用服务

#### 1. WWW (万维网)

- **核心协议：** **HTTP** (超文本传输协议)。
    
- **核心语言：** **HTML** (超文本标记语言)，用于编写网页文档。
    
- **工作原理：** 浏览器解析 HTML 并通过 HTTP 从 Web 服务器（如 Nginx, Apache）获取资源。
    

#### 2. 电子邮件 (E-mail)

- **SMTP (简单邮件传输协议)：** 用于发送邮件。
    
- **POP3/IMAP：** 用于接收和读取邮件。
    

#### 3. 文件传输协议 (FTP)

- 用于在 Internet 上进行文件的上传与下载，支持交互式访问。
    

---

### 💡 核心考点总结

- **IP 地址与 MAC 地址的区别：** IP 是网络层逻辑地址，MAC 是数据链路层物理地址。
    
- **B/S vs C/S：** B/S 跨平台性好，客户端无需安装特定软件；C/S 性能通常更高，交互更复杂。
    
- **邮件协议区分：** 发邮件找 **SMTP**，收邮件找 **POP3**。
