### 模块一：Vim 编辑器核心操作

Vim 是 Linux 环境下最核心的纯命令行编辑器，拥有无图形界面依赖、全快捷键操作等优势 。它的核心在于**三种模式的切换** 。

#### 1. 正常模式 (Normal Mode)

打开文件的默认模式，用于移动光标、复制、粘贴和删除 。

- **模式切换**：从其他模式按 `Esc` 键返回 。
- **光标移动**：
    
    - `h` (左), `j` (下), `k` (上), `l` (右) 。
    - `0` 或 `Home` (行首)，`$` 或 `End` (行尾) 。
    - `gg` (跳至第一行)，`G` (跳至最后一行)，`nG` (跳至第 n 行，如 `1G`) 。

- **文本操作**：
    
    - **复制/粘贴**：`yy` (复制当前行)，`nyy` (复制 n 行)；`p` (在光标下一行粘贴) 。
    - **删除/剪切**：`x` (删除单个字符)，`dd` (删除/剪切当前行)，`ndd` (删除 n 行) 。
    - **撤销**：`u` (撤销上一步操作) 。

#### 2. 编辑模式 (Insert Mode)

用于输入和修改纯文本 。

- **进入方式**：在正常模式下按 `i` (光标前插入)、`a` (光标后插入)、`o` (向下另起一行) 等 。

#### 3. 命令行模式 (Command-line Mode)

用于保存、退出、搜索和替换 。

- **进入方式**：在正常模式下输入 `:`或者 `/` 。

| **功能**    | **命令**              | **说明**                              |
| --------- | ------------------- | ----------------------------------- |
| **保存与退出** | `:w`                | 保存文件 。                              |
|           | `:q`                | 退出（未修改时） 。                          |
|           | `:wq`               | 保存并退出 。                             |
|           | `:q!`               | 强制退出且不保存 。                          |
| **环境设置**  | `:set nu`           | 显示行号 。                              |
| **搜索**    | `/word`             | 向下搜索 "word"。按 `n` 找下一个，按 `N` 找上一个 。 |
| **替换**    | `:%s/word1/word2/g` | 将全文中的 word1 替换为 word2 。             |

(注：Vim 还有可视块模式 `Ctrl+V` 用于选中区块，以及多窗口操作如 `vim -p` 和丰富的插件系统如 NERDTree 。)

### 模块二：Shell 脚本编程基础

Shell 脚本（通常以 `.sh` 结尾）是利用 bash 解释器批量执行 Linux 命令的自动化工具，非常适合科研和超算任务提交 。

#### 1. 变量的定义与使用

- **定义规则**：`变量名=值`。**等号两侧绝对不能有空格** 。不能以数字开头 。
- **调用变量**：使用 `$` 符号，例如 `echo $PATH` 。
- **常用环境变量**：`$PATH` (可执行文件路径)、`$PWD` (当前目录)、`$USER` (当前用户) 。
>[!WARNING]
>==`变量名=值`**等号两侧绝对不能有空格！**==
#### 2. 三种引号的核心区别（重要考点）

- **单引号 `''`**：最强屏蔽，所见即所得，内部的变量 `$var` 不会被解析 。
- **双引号 `""`**：温和解析，会解析内部的变量值 。
- **反引号 `` ` ` ``**：命令替换，会先执行里面的命令，再把结果输出出来 。

>[!NOTE]
>`$()`和反引号可以等价替换，都是完全解析，执行命令
#### 3. 运算符

- **算术运算**：
	- 使用 `$((运算式))` 进行整数加减乘除。
	- 浮点数需借助 `bc` 命令（如 `echo "10/3" | bc`） 。

- **比较与逻辑运算 (常用于 if 语句)**：
    
    - **比较**：`-eq` (等于), `-ne` (不等), `-gt` (大于), `-lt` (小于), `-ge` (大于等于), `-le` (小于等于) 。
    - **逻辑**：`-a` (与 AND), `-o` (或 OR), `!` (非 NOT) 。

>[!WARNING]
\>使用中括号 `[ ]` 时，括号内侧两边必须有空格 。

#### 4. 流程控制语句

- **条件判断 (if-else)**:
- ```sh
    if [ 条件 ]; then
        执行语句
    elif [ 条件 ]; then
        执行语句
    else
        执行语句
    fi
    ```
- **循环语句 (for/while)**:
**for 循环** (适合已知范围遍历):
```sh
    for i in {1..5}; do
        执行语句 (例如 mkdir calc_$i)
    done
```
**while 循环** (适合条件成立时持续执行):
```sh
	while [ 条件 ]; do
	   执行语句
	done
```
#### 5.读入
使用`read`指令
```sh
#!/bin/bash
#多条件判断整数范围
read -p "请输入一个数字：" num  # read：从终端读取用户输入，赋值给nun
if [ $num -gt 100 l ]; then
	echo "你输入的数字大于100"
elif [ $num -gt 50 l ]; then
	echo "你输入的数字大于5o目小于等于100"
else 
	echo "你输入的数字小于等于5θ"
fi
```
