## 一、 ⏱️ 时间测量 `<ctime>`

用于评估算法运行效率和性能。

|**功能**|**代码语法**|**说明**|
|---|---|---|
|**获取当前时间**|`time_t curr_time = time(NULL);`|获取系统当前时间戳 。|
|**转为字符输出**|`char *tm = ctime(&curr_time);`|将 `time_t` 格式转为可打印的字符串格式 。|
|**计算耗时**|`difftime(time_finish, time_start);`|计算两个时间点之间的时间差 。|
|**程序休眠**|`sleep(2);`|暂停程序运行（需包含 `<unistd.h>`，仅限 Linux/macOS） 。|

## 二、 🔤 字符串操作 `<string>`

`std::string` 是动态分配内存的，相比 C 风格字符数组（以 `\0` 结尾且大小固定）更安全灵活 。

|**操作**|**代码语法**|**说明**|
|---|---|---|
|**获取长度**|`s1.length()`|获取字符串字符数 。|
|**查找字符/子串**|`s1.find(s2, pos)`|从 `pos` 位置开始查找 `s2`，返回索引。若没找到会返回超出长度的值 。|
|**截取子串**|`s1.substr(start, len)`|从 `start` 位置截取 `len` 个字符 。|
|**擦除字符**|`s1.erase(start, len)`|擦除从 `start` 开始的 `len` 个字符。若不填 `len` 则擦除到底 。|
|**插入字符**|`s1.insert(pos, "str")`|在 `pos` 位置插入字符串 。|

## 三、 🔒 常量限定符 `const`

用于保护变量不被意外修改，提高代码安全性与可读性 。

- **基础用法：** `const int i = 10;` （定义后不可修改） 。
- **限定规律：** 先作用在 `const` 的左边，如果没有左边，再作用到右边 。
- **函数传参保护：** 推荐在函数引用传参时使用，如 `void Foo(const string &in);`，防止函数内部无意间修改原变量 。

## 四、 🖥️ 屏幕输入输出与格式化 `<iostream>` & `<iomanip>`

C++ 将 I/O 视为字节流，利用缓冲区进行批量传输 。

#### 1. 缓冲区避坑 (Buffer)

- **机制：** `cout` 是把数据先送入缓冲区，而不是直接上屏幕 。
- **刷新缓冲区：** `cout << endl;` 会插入换行并**立即刷新缓冲区**。
- **⚠️ Debug 避坑：** 程序崩溃（如数组溢出）时，缓冲区内的数据可能还未输出就死亡了。用 `endl` 强制刷新，能确保你看到崩溃前最后一条准确的输出 。

#### 2. 格式化控制 (Manipulators)

需引入 `<iomanip>` 头文件 。

|**格式控制**|**代码语法**|**说明**|
|---|---|---|
|**设置小数位数**|`cout << setprecision(3)`|设置有效数字（或小数位数）为 3 。|
|**固定小数输出**|`setiosflags(ios::fixed)`|以固定小数位数输出 。|
|**科学计数法**|`setiosflags(ios::scientific)`|以科学计数法输出 。|
|**设置位宽**|`setw(10)`|设置下一个输出占据 10 个字符的空间 。|
|**设置填充符**|`setfill('#')`|空白位置用 `#` 填充 。|
|**输出十六进制**|`hex`|以 16 进制输出数字 。|

## 五、 📁 文件输入输出 `<fstream>`

读写文件需要创建流对象，并将对象与特定文件连接 。

#### 1. 文本读写

- **读入文件 (Input)：** `ifstream ifs("filename.dat");` 。
- **写出文件 (Output)：** `ofstream ofs("filename.dat");` 。
- **追加模式写入：** `ofstream ofs("filename.dat", ios_base::app);` 。
- **关闭文件：** 操作完成后记得调用 `ifs.close();` 或 `ofs.close();` 。

2. 二进制文件读写 (更精确、更快、更小)

- **写二进制：** `ofs.write((char*)&data, sizeof(data));` （需强制转换指针类型） 。
- **读二进制：** `ifs.read((char*)&data, sizeof(data));` 。

#### 3. 文件指针/光标控制

- **忽略特定内容：** `ifs.ignore(150, '\n');` （忽略 150 个字符或直到遇到换行符） 。
- **移动读取指针：** `ifs.seekg(0);` （将读取光标挪回到文件开头） 。
- **移动写入指针：** `ofs.seekp(...)` （挪动输出光标位置） 。

## 六、 🔄 字符串流 `<sstream>`

非常适合用来动态拼接生成文件名，或从字符串中提取信息 。

1. **定义对象：** `stringstream ssm;`
2. **拼接内容：** `ssm << "data_" << index << ".txt";`
3. **类型转换：**
    - 转为 C++ 字符串： `ssm.str()`
    - 转为 C 风格指针： `ssm.str().c_str()` （部分旧的文件操作函数需要此格式）
