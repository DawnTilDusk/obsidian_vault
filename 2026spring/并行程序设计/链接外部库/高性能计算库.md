## 1. 常用高性能计算库概述

在科学计算和工程计算中，调用经过高度优化的底层数学库能大幅提升计算效率 。

- **BLAS (Basic Linear Algebra Subprograms)**：基础线性代数程序集，主要用于向量和矩阵的基础操作 。它分为三级：
    
    - **第一级**：基本向量操作（如向量加法、内积等） 。
    - **第二级**：矩阵与向量的乘法等操作 。
    - **第三级**：复杂的矩阵运算（如矩阵分解、求逆等） 。
    
    - _注_：**OpenBLAS** 是一个开源且高性能的 BLAS 实现，针对现代多核 CPU 进行了高度并行优化，性能远超传统单核 BLAS 。
    
- **LAPACK (Linear Algebra PACKage)**：建立在 BLAS 之上的高级线性代数库 。它提供更复杂的算法，如线性方程组求解、特征值计算和矩阵分解 。LAPACK 的安装包中通常已经包含了 BLAS 和 LAPACKE（LAPACK 的 C 语言接口） 。

- **FFTW (Fastest Fourier Transform in the West)**：用于计算离散傅里叶变换（DFT）的标准 C 语言程序集 。利用对称性和周期性，它能将 DFT 的时间复杂度从 $O(N^2)$ 降低到 $O(N \log N)$ 。

## 2. 静态链接库与动态链接库

在 C++ 开发中调用外部库时，理解静态库和动态库的区别至关重要 。

|**特性**|**静态库 (.a)**|**动态库 (.so / .dll)**|
|---|---|---|
|**工作原理**|把代码“抄进”你的程序里|让程序运行时去外面“找代码”用|
|**链接时机**|编译/链接时|程序运行时|
|**文件大小**|生成的可执行文件较大|生成的可执行文件较小|
|**依赖关系**|无外部依赖，独立运行|运行时必须存在该库文件|
|**更新方式**|库更新后，需重新编译程序|直接替换库文件即可生效|

## 3. 编译与安装速查

### 方式一：极速安装 (Ubuntu / Debian)

使用 `apt` 包管理器可一键安装，文件会自动存放到系统默认目录 ：

```sh
sudo apt update
sudo apt install libopenblas-dev libblas-dev liblapack-dev libfftw3-dev
```

### 方式二：源码编译安装

如果需要自定义优化参数或安装目录，可以采用源码编译 。

- **配置环境变量**：为了在编译时不反复写长路径，建议在 `~/.bashrc` 中配置静态库路径（`LIBRARY_PATH`）和头文件路径（`C_INCLUDE_PATH`） 。

- **编译指令参数**：在使用 `g++` 编译时，使用 `-L` 指定额外的库文件搜索路径，使用 `-l` 指定要链接的具体库名称 。

## 4. 核心 API 与代码实战

### 实战 1：BLAS 矩阵乘法 (`cblas_dgemm`)

- **数学公式**：$C = \alpha AB + \beta C$ 。

- **参数速记**：`Order` 决定存储顺序（如 `CblasRowMajor` 为行优先），`TransA`/`TransB` 决定是否转置，`M`, `N`, `K` 分别对应矩阵的维度 。

```cpp
#include <cblas.h>

// 计算 C = 1.0 * A * B + 0.0 * C
cblas_dgemm(
    CblasRowMajor, CblasNoTrans, CblasNoTrans, 
    m, n, k, 1.0, a, k, b, n, 0.0, c, n
);
```

- **编译**：`g++ link_blas.cpp -L"/path/to/OpenBLAS" -lopenblas` 。

### 实战 2：LAPACK 矩阵对角化 (`dsyev` / `zheev`)

- **`dsyev`**：用于计算**实对称矩阵**的特征值和特征向量 。
- **`zheev`**：用于计算**复数厄密矩阵**的对角化 。输入矩阵需要使用 `lapack_complex_double` 类型构建 。

```cpp
#include <lapacke.h>

// 实对称矩阵对角化
// 'V' 表示同时计算特征值和特征向量，'U' 表示读取上三角矩阵数据
int info = LAPACKE_dsyev(LAPACK_ROW_MAJOR, 'V', 'U', n, a, lda, w);

if (info != 0) {
    cout << "Failed to diagonalize matrix." << endl;
}
```

### 实战 3：FFTW 一维快速傅里叶变换

- **核心流程**：分配内存 -> 创建 Plan -> 执行 Plan -> 销毁 Plan 并释放内存 。
- **特别注意**：`fftw_plan_dft_r2c_1d` 专用于实数序列到复数序列的转换 。

```cpp
#include <fftw3.h>

double *in = new double[n](); // 实数输入
fftw_complex out[n / 2 + 1];  // 复数输出

// 1. 创建计划
fftw_plan p = fftw_plan_dft_r2c_1d(n, in, out, FFTW_ESTIMATE);

// 2. 执行变换
fftw_execute(p);

// 3. 善后处理
fftw_destroy_plan(p);
fftw_cleanup();
delete[] in;
```
