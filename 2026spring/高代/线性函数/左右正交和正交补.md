## 概念一览表

在往下看之前，请死死记住我们的大前提公式（三明治公式）：

$$f(\alpha, \beta) = x^T A y$$

_($x$ 是左向量坐标，横着撞向 $A$；$y$ 是右向量坐标，竖着被 $A$ 撞)_

| **讲义里的“高冷”概念**         | **对应的“大白话”**      | **对应的“矩阵本质” (最强必杀技)**                               |
| ---------------------- | ----------------- | --------------------------------------------------- |
| **左向量 $\alpha$**       | “主动撞击的雷达波”        | 行向量 $x^T$，作用于 $A$ 的**行空间**                          |
| **右向量 $\beta$**        | “被动接受探测的飞机”       | 列向量 $y$，作用于 $A$ 的**列空间**                            |
| **右核空间 $\ker_R f$**    | “对所有雷达都隐形的飞机”     | 矩阵 $A$ 的**零空间 $\text{Nul}(A)$** （满足 $Ay = 0$）       |
| **左核空间 $\ker_L f$**    | “探不到任何飞机的瞎子雷达”    | 矩阵 $A$ 的**左零空间 $\text{Nul}(A^T)$** （满足 $A^T x = 0$） |
| **右正交补 $W^{\perp_R}$** | “对特定机群 $W$ 隐形的飞机” | 矩阵组 $(X^T A)$ 的**零空间**                              |
| **左正交补 $W^{\perp_L}$** | “探不到特定机群 $W$ 的雷达” | 矩阵组 $(A Y)$ 的**左零空间**                               |

## 联系

#### 1. 左右核空间，就是你在找的“零空间与左零空间”

正如你猜想的，既然 $f(\alpha, \beta) = x^T A y$，那么：

- **什么是右核？** 就是让所有左边来的 $x^T$ 乘上去都变 0。那必然要求 $(Ay) = \mathbf{0}$。这不就是你大一学的求解齐次线性方程组 $Ay=0$ 吗？找出的所有解 $y$ 构成的空间，就是**零空间（Null Space）**。

- **什么是左核？** 就是让所有右边来的 $y$ 乘上去都变 0。那必然要求 $(x^T A) = \mathbf{0}^T$。转置一下就是 $A^T x = 0$。找出来的解 $x$，就是大名鼎鼎的**左零空间（Left Null Space）**。

- **维度为什么相等？** 因为矩阵 $A$ 的秩等于 $A^T$ 的秩！既然“总维度 - 秩 = 零空间维度”，那这俩的维度必然死死绑定，一模一样大（命题 6.10）。

### 2. 正交补空间：“换个新矩阵求零空间”

如果把 $W$ 的基拼成一个矩阵 $B$（此时 $W = \text{Col}(B)$），且双线性函数的度量矩阵为 $A$，那么：
- $W$ 的**右正交补**，是矩阵 $B^T A$ 的零空间。
- $W$ 的**左正交补**，是矩阵 $B^T A^T$ 的零空间。
#### 1） W 的右正交补 ($W^{\perp_R}$)

我们要找所有的 $y$，使得它被所有的 $w$ 从左边撞击时结果为 0：

$$w^T A y = 0$$

把 $w = Bu$ 代入：

$$(Bu)^T A y = 0 \implies u^T (B^T A y) = 0$$

因为这个式子对任意的 $u$ 都得成立，所以括号里的东西必须是一坨 0：

$$B^T A y = 0$$

**终极结论：W 的右正交补，是复合矩阵 $(B^T A)$ 的零空间 $\text{Nul}(B^T A)$。**

#### 2） W 的左正交补 ($W^{\perp_L}$)

我们要找所有的 $x$，使得它从左边撞击所有的 $w$ 时结果为 0：

$$x^T A w = 0$$

把 $w = Bu$ 代入：

$$x^T A (Bu) = 0 \implies (x^T A B) u = 0$$

同理，这要求前面的行向量全是 0：

$$x^T A B = \mathbf{0}^T$$

两边同时取转置，把它竖起来变成我们熟悉的求零空间形式：

$$B^T A^T x = 0$$

**终极结论：W 的左正交补，是复合矩阵 $(B^T A^T)$ 的零空间 $\text{Nul}(B^T A^T)$。**

### 3. 神奇的维度守恒，就是“秩-零化度定理”

讲义里最后一个不加证明的命题 6.11 说：$\dim W + \dim W^{\perp_R} = n$。

在非退化（$A$ 是满秩的可逆方阵）的前提下，我们用矩阵理论直接秒杀它：

既然求 $W^{\perp_R}$ 就是求矩阵 $(X^T A)$ 的零空间，那么根据**秩-零化度定理**：

$$\text{零空间维度} = \text{总维度 } n - \text{矩阵的秩}$$

$$\dim W^{\perp_R} = n - \text{rank}(X^T A)$$

因为 $A$ 是满秩可逆阵，它乘在 $X^T$ 后面**绝对不会改变秩**，所以 $\text{rank}(X^T A) = \text{rank}(X^T)$。

而 $X^T$ 的秩，就是子空间 $W$ 的维度 $\dim W$。

代入进去：

$$\dim W^{\perp_R} = n - \dim W$$

瞬间证毕！这哪里是抽象代数，这明明就是齐次线性方程组的维度公式换了套马甲而已。
