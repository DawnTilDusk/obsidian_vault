## 1. H特征多项式就是构造H的多项式

如果我凭空给你一个多项式 $f(\lambda) = \lambda^m + h_{m-1}\lambda^{m-1} + \dots + h_1\lambda + h_0$。

然后我按照上一张 PPT 的规则，把系数填到最后一列，左下角铺满 1，构造出一个友阵 $H$。

奇迹在于，如果你去手算这个友阵的特征多项式 $|\lambda I - H|$（比如按最后一行或最后一列展开），你会发现：

$$|\lambda I - H| \equiv f(\lambda)$$

## 2. H的**最小多项式也等于 $f(\lambda)$**

---

你已经知道了友阵 $H$ 是根据多项式 $f(\lambda)$ 的系数“捏造”出来的，但凭什么这个拼凑出来的矩阵，它的最小多项式刚好就是 $f(\lambda)$ 呢？

这并非巧合，而是一个极其严密且漂亮的代数必然。要证明这一点，我们只需要在友阵自身所处的标准坐标系里，完成两步非常干脆的逻辑推演：**证明它是零化多项式**，以及**证明它必定是“最小”的**。

设我们根据多项式 $f(\lambda) = \lambda^m + h_{m-1}\lambda^{m-1} + \dots + h_1\lambda + h_0$ 构造出了一个 $m \times m$ 的友阵 $H$。

在这个 $m$ 维空间里，我们取最朴素的标准正交基：

$$\mathbf{e}_1 = \begin{bmatrix} 1 \\ 0 \\ \vdots \\ 0 \end{bmatrix}, \quad \mathbf{e}_2 = \begin{bmatrix} 0 \\ 1 \\ \vdots \\ 0 \end{bmatrix}, \quad \dots, \quad \mathbf{e}_m = \begin{bmatrix} 0 \\ 0 \\ \vdots \\ 1 \end{bmatrix}$$

### 第一步：友阵的“移位魔法”

根据友阵 $H$ 的结构（主对角线下方全为 1，其余为 0，除了最后一列），如果你拿 $H$ 去不断乘以第一个基向量 $\mathbf{e}_1$，你会发现它就是一个完美的“平移机器”：

$$H\mathbf{e}_1 = \mathbf{e}_2$$

$$H^2\mathbf{e}_1 = H(H\mathbf{e}_1) = H\mathbf{e}_2 = \mathbf{e}_3$$

$$\vdots$$

$$H^{m-1}\mathbf{e}_1 = \mathbf{e}_m$$

但当你乘到第 $m$ 次时，由于碰到的是最后一列，平移结束，规则变了：

$$H^m\mathbf{e}_1 = H(H^{m-1}\mathbf{e}_1) = H\mathbf{e}_m = \begin{bmatrix} -h_0 \\ -h_1 \\ \vdots \\ -h_{m-1} \end{bmatrix} = -h_0\mathbf{e}_1 - h_1\mathbf{e}_2 - \dots - h_{m-1}\mathbf{e}_m$$

### 第二步：证明 $f(H) = \mathbf{O}$（它是零化多项式）

我们把刚才最后一步得到的等式移项，把所有项都放到等号左边：

$$H^m\mathbf{e}_1 + h_{m-1}\mathbf{e}_m + \dots + h_1\mathbf{e}_2 + h_0\mathbf{e}_1 = \mathbf{0}$$

再把之前推导的 $\mathbf{e}_k = H^{k-1}\mathbf{e}_1$ 替换回去：

$$H^m\mathbf{e}_1 + h_{m-1}H^{m-1}\mathbf{e}_1 + \dots + h_1H\mathbf{e}_1 + h_0I\mathbf{e}_1 = \mathbf{0}$$

把 $\mathbf{e}_1$ 提取出来：

$$(H^m + h_{m-1}H^{m-1} + \dots + h_1H + h_0I)\mathbf{e}_1 = \mathbf{0}$$

也就是：

$$f(H)\mathbf{e}_1 = \mathbf{0}$$

好，既然超级矩阵 $f(H)$ 能把 $\mathbf{e}_1$ 变成零，那其他基向量呢？

因为矩阵乘法满足交换律，对于任意基向量 $\mathbf{e}_k = H^{k-1}\mathbf{e}_1$，我们有：

$$f(H)\mathbf{e}_k = f(H)(H^{k-1}\mathbf{e}_1) = H^{k-1}(f(H)\mathbf{e}_1) = H^{k-1}\mathbf{0} = \mathbf{0}$$

**结论 1：** $f(H)$ 把空间里所有的基向量都变成了零，所以 $f(H)$ 彻头彻尾就是一个零矩阵 $\mathbf{O}$。因此，$f(\lambda)$ 确实是 $H$ 的一个零化多项式。

---

### 第三步：证明它是“最小”的（核心杀招）

“最小多项式”不仅要求能零化矩阵，还要求它的**次数必须是最低的**。

我们用反证法：假设存在一个次数比 $m$ 还要低（假设为 $k$ 次，其中 $k < m$）的多项式 $g(\lambda)$，它也能零化 $H$。

设 $g(\lambda) = a_k\lambda^k + \dots + a_1\lambda + a_0$。

既然 $g(H) = \mathbf{O}$，那么它作用在 $\mathbf{e}_1$ 上必然也是零：

$$g(H)\mathbf{e}_1 = a_kH^k\mathbf{e}_1 + \dots + a_1H\mathbf{e}_1 + a_0\mathbf{e}_1 = \mathbf{0}$$

同样，利用前面的“平移魔法” $H^i\mathbf{e}_1 = \mathbf{e}_{i+1}$ 替换进去：

$$a_k\mathbf{e}_{k+1} + \dots + a_1\mathbf{e}_2 + a_0\mathbf{e}_1 = \mathbf{0}$$

**破绽出现了！**

等式左边是标准基向量 $\mathbf{e}_1, \mathbf{e}_2, \dots, \mathbf{e}_{k+1}$ 的线性组合。

因为这些标准基向量是**绝对线性无关**的，它们组合起来等于零向量的唯一可能，就是所有的系数全部等于零：

$$a_k = 0, \dots, a_1 = 0, a_0 = 0$$

这意味着，如果一个次数小于 $m$ 的多项式想要零化 $H$，它只能是一个所有的系数全是 0 的“零多项式”。

**结论 2：** 在达到 $m$ 次之前，没有任何一个非零多项式能干掉 $H$。这就从根本上锁死了 $f(\lambda)$ 的地位——它不仅是零化多项式，而且必定是次数最低的那一个。

---

这就是代数结构里的“锁扣”设计：

友阵左下角那排负责平移的 1，保证了 $\mathbf{e}_1, H\mathbf{e}_1, \dots, H^{m-1}\mathbf{e}_1$ 绝对线性无关（排除了低次多项式）；而最后一列的系数，刚好在第 $m$ 步完成了完美的抵消（确立了 $m$ 次多项式的存在）。两头一堵，$H$ 的最小多项式就死死地绑定为 $f(\lambda)$ 了。
