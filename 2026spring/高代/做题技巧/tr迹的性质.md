矩阵的迹（Trace，记作 $\text{tr}(A)$）是线性代数中一个极其优美且实用的工具。对于一个 $n$ 阶方阵 $A$，它的迹定义为主对角线元素的总和：$\text{tr}(A) = \sum_{i=1}^n a_{ii}$。

掌握它的性质，在做理论证明（如刚才的几道题）或是后续工科的矩阵计算中都能起到四两拨千斤的作用。以下为你系统总结迹的核心性质：

### 1. 基础代数性质（线性与转置）

- **线性性质**：迹算子是一个线性映射（正如你在 4.8.5 题中证明的那样）。对于任意常数 $c, d$ 和同阶方阵 $A, B$：
    
    $$\text{tr}(cA + dB) = c\,\text{tr}(A) + d\,\text{tr}(B)$$
    
- **转置不变性**：矩阵转置后，主对角线上的元素位置不发生改变，因此：
    
    $$\text{tr}(A^T) = \text{tr}(A)$$
    
    如果复数矩阵取共轭转置（Hermitian共轭 $A^H$），则 $\text{tr}(A^H) = \overline{\text{tr}(A)}$。
    

### 2. 乘法与交换性质（极其重要）

- **乘法交换律（轮换对称性）**：这是迹最“好用”的性质之一（也是刚才证明 $\mathscr{AB}-\mathscr{BA} \neq \mathscr{E}$ 的核心）。如果 $A$ 是 $m \times n$ 矩阵，$B$ 是 $n \times m$ 矩阵，那么：$$\text{tr}(AB) = \text{tr}(BA)$$
- **循环移位不变性**：对于多个矩阵的乘积，迹在整体循环移位下保持不变。例如对三个方阵：$$\text{tr}(ABC) = \text{tr}(BCA) = \text{tr}(CAB)$$
    **⚠️ 避坑指南：** 迹只能**循环**移位，不能任意交换。一般情况下 $\text{tr}(ABC) \neq \text{tr}(BAC)$。

### 3. 与特征值和多项式的深层联系

- **特征值之和**：矩阵的迹等于其所有复特征值（按代数重数计）的总和。$$\text{tr}(A) = \sum_{i=1}^n \lambda_i$$
    _(与之遥相呼应的是：矩阵的行列式等于所有特征值之积 $\det(A) = \prod_{i=1}^n \lambda_i$)_。

- **相似不变性**：如果矩阵 $A$ 和 $B$ 相似（即存在可逆矩阵 $P$ 使得 $B = P^{-1}AP$），那么它们的迹相等：$$\text{tr}(B) = \text{tr}(P^{-1}AP) = \text{tr}(A(PP^{-1})) = \text{tr}(A)$$
    这说明迹是线性变换的固有属性，不依赖于基的选择。

- **特征多项式系数**：在矩阵 $A$ 的特征多项式 $f(\lambda) = \det(\lambda I - A) = \lambda^n + c_1\lambda^{n-1} + \dots + c_n$ 中，次高项的系数总是等于迹的相反数：
    
    $$c_1 = -\text{tr}(A)$$
    

### 4. 进阶应用（在物理与工程中常用）

- **Frobenius 内积（矩阵内积）**：在研究机器人学或机器学习时，经常需要衡量两个矩阵的差异，这时候会用到迹来定义矩阵空间上的内积：
    
    $$\langle A, B \rangle = \text{tr}(A^T B)$$
    
- **幂零矩阵的判定**：如果对于所有的 $k = 1, 2, \dots, n$，都有 $\text{tr}(A^k) = 0$，那么 $A$ 是一个幂零矩阵（即存在某个整数 $m$ 使得 $A^m = O$）。
