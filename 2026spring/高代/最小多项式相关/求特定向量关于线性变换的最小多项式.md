这类求「特定向量关于线性变换的最小多项式」的题目，是高等代数/线性代数中非常核心的考点。它本质上是在探究一个几何问题：**这个向量在不断被线性变换 $\mathscr{A}$ 作用后，生成的轨迹空间（Krylov 子空间）维数是多大？**

遇到这类题，通常有两套标准的“武功秘籍”。面对你图里的这种带具体几何意义（旋转）的题，第二套方法是降维打击。

### 总结：求解向量最小多项式的两大流派

#### 战术一：定义法（Krylov 序列法）—— 适合“不知底细”的抽象矩阵

**核心思想：硬算。** 最小多项式 $m_\alpha(\lambda)$ 的次数，就是序列 $\alpha, \mathscr{A}(\alpha), \mathscr{A}^2(\alpha) \dots$ 第一次出现线性相关时的项数。

- **步骤：**
    
    1. 计算 $\alpha$。
        
    2. 计算 $\mathscr{A}(\alpha)$，看它是否与 $\alpha$ 线性相关。如果是（即 $\mathscr{A}(\alpha) = k\alpha$），则最小多项式为 $\lambda - k$。
        
    3. 如果无关，继续算 $\mathscr{A}^2(\alpha)$，看它能否由 $\alpha$ 和 $\mathscr{A}(\alpha)$ 线性表出。如果 $\mathscr{A}^2(\alpha) = c_1\mathscr{A}(\alpha) + c_0\alpha$，则最小多项式为 $\lambda^2 - c_1\lambda - c_0$。
        
    4. 依此类推，直到找到线性关系。
        

#### 战术二：空间分解法（利用不变子空间）—— 适合本题这类“结构清晰”的变换

**核心思想：化整为零。** 整个空间的最小多项式是全集的“天花板”，单个向量的最小多项式必须整除它。

- **步骤：**
    
    1. **求全局：** 求出整个线性变换 $\mathscr{A}$ 的最小多项式 $m(\lambda)$。
        
    2. **拆空间：** 根据 $m(\lambda)$ 的因式分解，将全空间 $V$ 直和分解为几个不变子空间 $V = W_1 \oplus W_2 \oplus \dots$
        
    3. **拆向量：** 将待求向量 $\alpha$ 投影到这些子空间上，即 $\alpha = \alpha_1 + \alpha_2 + \dots$
        
    4. **求 LCM：** 向量 $\alpha$ 的最小多项式，就是它所在各个非零分量 $\alpha_i$ 的最小多项式的**最小公倍式（LCM）**。
        

### 实战演练：用“战术二”秒杀你的题目

![[Pasted image 20260605173525.png]]

题目背景：三维空间绕 $\boldsymbol{\varepsilon}_1$ 轴旋转 $\theta$ 角（$\theta \neq k\pi$）。

**第一步：摸清 $\mathscr{A}$ 的全局底细**

在基 $\boldsymbol{\varepsilon}_1, \boldsymbol{\varepsilon}_2, \boldsymbol{\varepsilon}_3$ 下，旋转变换的矩阵是：

$$A = \begin{pmatrix} 1 & 0 & 0 \\ 0 & \cos\theta & -\sin\theta \\ 0 & \sin\theta & \cos\theta \end{pmatrix}$$

它的特征多项式为 $f(\lambda) = (\lambda - 1)(\lambda^2 - 2\cos\theta \lambda + 1)$。

因为转角不是 $\pi$ 的倍数，所以方程 $\lambda^2 - 2\cos\theta \lambda + 1 = 0$ 没有实根 $1$ 或 $-1$（它是一对共轭复根 $e^{\pm i\theta}$）。

特征根互不相同，所以全局的最小多项式就等于特征多项式：

$$m(\lambda) = (\lambda - 1)(\lambda^2 - 2\cos\theta \lambda + 1)$$

**第二步：空间直和分解**

全空间自然裂解为两个不变子空间的直和 $V = W_1 \oplus W_2$：

- **转轴空间 $W_1 = \text{span}(\boldsymbol{\varepsilon}_1)$：** 这是一个 1 维空间，在这个空间里所有的向量都不动，因此 $W_1$ 上的最小多项式是 $m_1(\lambda) = \lambda - 1$。
    
- **旋转平面 $W_2 = \text{span}(\boldsymbol{\varepsilon}_2, \boldsymbol{\varepsilon}_3)$：** 这是一个 2 维空间，每次旋转都会改变方向，因此 $W_2$ 上任意非零向量的最小多项式都是 $m_2(\lambda) = \lambda^2 - 2\cos\theta \lambda + 1$。
    

**第三步：分头击破各个向量**

1. **对于 $\boldsymbol{\varepsilon}_1$：**
    
    它纯粹活在转轴空间 $W_1$ 里（在 $W_2$ 上分量为 0）。
    
    所以它的最小多项式就是：**$m_{\boldsymbol{\varepsilon}_1}(\lambda) = \lambda - 1$**。
    
    _(物理直觉：转轴上的向量转一次就不动了，$\mathscr{A}\boldsymbol{\varepsilon}_1 = 1\cdot\boldsymbol{\varepsilon}_1$)_
    
2. **对于 $\boldsymbol{\varepsilon}_3$：**
    
    它纯粹活在旋转平面 $W_2$ 里（在 $W_1$ 上分量为 0）。
    
    所以它的最小多项式直接继承 $W_2$ 的性质：**$m_{\boldsymbol{\varepsilon}_3}(\lambda) = \lambda^2 - 2\cos\theta \lambda + 1$**。
    
    _(物理直觉：在平面上旋转，你需要转两次 $\mathscr{A}^2$ 才能把它和前两次的状态凑成线性相关)_
    
3. **对于 $\boldsymbol{\varepsilon}_1 + \boldsymbol{\varepsilon}_2$：**
    
    这个向量“跨界”了。它在 $W_1$ 中的分量是 $\boldsymbol{\varepsilon}_1$（对应 $m_1(\lambda)$），在 $W_2$ 中的分量是 $\boldsymbol{\varepsilon}_2$（对应 $m_2(\lambda)$）。
    
    因为这两个分量都不为零，所以整个向量的最小多项式是它们各自最小多项式的最小公倍数（LCM）。
    
    由于 $\lambda - 1$ 和 $\lambda^2 - 2\cos\theta \lambda + 1$ 互素：
    
    **$m_{\boldsymbol{\varepsilon}_1 + \boldsymbol{\varepsilon}_2}(\lambda) = (\lambda - 1)(\lambda^2 - 2\cos\theta \lambda + 1)$**。
    
    _(物理直觉：这个向量斜指在空间中，旋转它会扫出一个完整的 3D 圆锥面，它的轨迹张成了完整的 3 维空间，所以必须用满次数的 3 次多项式才能将其归零)_
    
4. **对于 $\boldsymbol{\varepsilon}_2 + \boldsymbol{\varepsilon}_3$：**
    
    虽然它是两个向量的和，但这两个向量都属于旋转平面 $W_2$。所以它们的和依然完全躺在 $W_2$ 这个不变子空间里。
    
    只要在 $W_2$ 里，且不是零向量，它的最小多项式就必然是：**$m_{\boldsymbol{\varepsilon}_2 + \boldsymbol{\varepsilon}_3}(\lambda) = \lambda^2 - 2\cos\theta \lambda + 1$**。
