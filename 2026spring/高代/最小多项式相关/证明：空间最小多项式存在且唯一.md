### **1. 证明最小多项式的“存在性”**

要证明存在，我们需要说明**一定存在一个非零多项式能够让变换 $\mathscr{A}$ 化为零变换**。

1. **利用维数构造线性相关：**
    
    已知 $V$ 是 $n$ 维线性空间，那么在 $V$ 上的所有线性变换构成的空间（通常记为 $\text{End}(V)$ 或 $L(V,V)$）的维数是 $n^2$。
    
    现在，考虑线性变换 $\mathscr{A}$ 的前 $n^2+1$ 个幂次：
    
    $$\mathscr{E}, \mathscr{A}, \mathscr{A}^2, \dots, \mathscr{A}^{n^2}$$
    
    由于这个序列中一共有 $n^2+1$ 个线性变换，而它们所在的空间维数只有 $n^2$。根据线性代数的基本原理，这 $n^2+1$ 个变换必然是**线性相关**的。
    
2. **构造化零多项式：**
    
    既然线性相关，就必然存在一组不全为零的数域 $\mathbb{P}$ 中的标量 $c_0, c_1, \dots, c_{n^2}$，使得：
    
    $$c_0\mathscr{E} + c_1\mathscr{A} + c_2\mathscr{A}^2 + \dots + c_{n^2}\mathscr{A}^{n^2} = \mathscr{O}$$
    
    我们构造一个多项式 $f(\lambda) = c_0 + c_1\lambda + c_2\lambda^2 + \dots + c_{n^2}\lambda^{n^2}$。
    
    显然，$f(\lambda)$ 是一个非零多项式，并且满足 $f(\mathscr{A}) = \mathscr{O}$（即 $f(\lambda)$ 是 $\mathscr{A}$ 的一个零化多项式）。
    
    _(注：这里也可以直接引用哈密顿-凯莱定理得出特征多项式 $f(\lambda)$ 满足 $f(\mathscr{A})=\mathscr{O}$，但用维数证明更具有普适性，不依赖行列式的展开。)_
    
3. **确定最小次数：**
    
    令集合 $S$ 为所有满足 $g(\mathscr{A}) = \mathscr{O}$ 的非零多项式 $g(\lambda)$ 的集合。
    
    因为 $f(\lambda) \in S$，所以 $S$ 不是空集。
    
    在 $S$ 中，多项式的次数是一组非负整数。根据自然数的良序性，集合 $S$ 中必然存在一个**次数最小**的多项式。
    
    设这个次数最小的多项式为 $m(\lambda)$，将其除以它的最高次项系数，得到一个**首一多项式**。
    
    这个首一多项式完全符合最小多项式的定义。因此，**存在性得证**。
    

### **2. 证明最小多项式的“唯一性”**

唯一性的证明通常采用**反证法**（或者假设有两个，证明它们相等）。

1. **假设存在两个最小多项式：**
    
    假设 $\mathscr{A}$ 有两个最小多项式 $m_1(\lambda)$ 和 $m_2(\lambda)$。
    
    根据最小多项式的定义，它们必须满足三个条件：
    
    - 都是首一多项式（最高次项系数为 $1$）。
        
    - 满足 $m_1(\mathscr{A}) = \mathscr{O}$ 和 $m_2(\mathscr{A}) = \mathscr{O}$。
        
    - 它们的次数必定相同（设它们的次数同为 $k$），因为它们都是次数最小的零化多项式。
        
2. **构造差值多项式：**
    
    令 $d(\lambda) = m_1(\lambda) - m_2(\lambda)$。
    
    由于 $m_1(\lambda)$ 和 $m_2(\lambda)$ 都是次数为 $k$ 的首一多项式，当它们相减时，最高次项 $\lambda^k$ 会被完全抵消。
    
    因此，差值多项式 $d(\lambda)$ 的次数必然严格小于 $k$，即 $\deg(d) < k$。
    
3. **推导矛盾：**
    
    现在我们把变换 $\mathscr{A}$ 代入 $d(\lambda)$ 中：
    
    $$d(\mathscr{A}) = m_1(\mathscr{A}) - m_2(\mathscr{A}) = \mathscr{O} - \mathscr{O} = \mathscr{O}$$
    
    这说明 $d(\lambda)$ 也是 $\mathscr{A}$ 的一个零化多项式。
    
    如果 $d(\lambda)$ 不是零多项式，那么我们就找到了一个次数小于 $k$ 的非零零化多项式。这直接与 $m_1(\lambda)$（和 $m_2(\lambda)$）是“次数最小”的前提产生**矛盾**！
    
4. **得出结论：**
    
    为了避免上述矛盾，唯一的可能是 $d(\lambda)$ 必须是**零多项式**，也就是多项式的每一项系数都为 $0$。
    
    即：$d(\lambda) \equiv 0 \implies m_1(\lambda) = m_2(\lambda)$。
    
    因此，最小多项式是唯一的。**唯一性得证**。
