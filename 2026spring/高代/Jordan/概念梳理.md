>[!概念关系]
>- 最高阶的不变因子就是最小多项式：$d_6(\lambda) = g(\lambda) = (\lambda^2 - 2\lambda + 2)^2(\lambda - 1)$。 
>- 所有不变因子的乘积（行列式因子）就是特征多项式：$D_6=d_1 d_2 d_3 d_4 d_5 d_6 = f(\lambda) = (\lambda^2 - 2\lambda + 2)^2(\lambda - 1)^2$。



### 🗺️ 第一部分：全景结构层次图（从宏观到微观）

你可以把整个矩阵的结构想象成一个俄罗斯套娃，从外到内依次是：

- **🌐 宏观层（整个矩阵）**
    
    - **特征多项式 $f(\lambda)$**：整个矩阵的“人口普查表”，记录了所有特征值以及它们的总人口（代数重数）。
    - **最小多项式 $m(\lambda)$**：整个矩阵的“天花板限制”，记录了所有特征值所在的最长链条长度。
    - **Jordan 标准型 ($J$)**：整个矩阵在最完美坐标系下的终极长相（所有积木拼在一起的定妆照）。

- **🏢 中观层（按特征值 $\lambda_i$ 划分的独立子空间）**
    - **广义特征空间 $V_i$**：属于特征值 $\lambda_i$ 的专属大房间。房间的总面积由**代数重数 $k_i$** 决定。

- **🧱 微观层（大房间内部的具体结构）**
	[[最小多项式和特征多项式关系]]
    - **特征空间 $\ker(A-\lambda_i I)$**：大房间里的“地基”。地基的数量由**几何重数 $g_i$** 决定（也就是真正特征向量的个数）。
    - **Jordan 链**：从地基（特征向量）往上建起的“承重墙”。有几个地基就有几条链。
    - **Jordan 块 $J_m(\lambda_i)$**：一条 Jordan 链在代数上写出来的样子。
    - **最小多项式的重数 $m_i$**：这个大房间里，**最高**的那堵承重墙的高度。
### 📖 第二部分：核心翻译字典（考试秒杀专区）

做题时，看到左边的词，立刻在脑子里翻译成右边的几何动作：

|**代数语言（题目告诉你的条件）**|**几何/物理直观（你脑子里要浮现的画面）**|**对应 Jordan 块的结构限制**|
|---|---|---|
|**代数重数 $k_i$** (特征多项式幂次)|**“积木的总块数”**|决定了该特征值下，所有单 Jordan 块的**阶数总和**。|
|**几何重数 $g_i$** (真正特征向量个数)|**“积木堆的个数”**（地基的数量）|决定了该特征值下，**单 Jordan 块的总个数**。|
|**最小多项式的重数 $m_i$**|**“最高那一堆积木的层数”**|决定了该特征值下，**最大的那个单 Jordan 块的阶数**。|
|**广义特征向量** (满足 $(A-\lambda I)^k v = 0$)|**“Jordan 链上的节点”**（水流从高往低流）|向量被算子作用一次，就顺着链条往下降一格，直到归零。|
|**可对角化**|**“全都是平房，没有楼房”**|最小多项式无重根 ($m_i=1$)，几何重数 = 代数重数 ($g_i=k_i$)。所有 Jordan 块都是 $1 \times 1$。|

### 🧩 第三部分：硬核串联逻辑（一段话打通任督二脉）

假设某矩阵有一个特征值 $\lambda = 2$，其**代数重数 $k=5$**（我们一共有5块积木）。

1. 这5块积木怎么分？要看**几何重数 $g$**（特征向量个数）。如果 $g=2$，说明只有2个真正的特征向量，这5块积木必须分成 2 堆。也就对应 **2 个 Jordan 块**。
    
2. 这 2 堆积木怎么摞？可能是 4+1，也可能是 3+2。这时候看**最小多项式的重数 $m$**。如果 $m=3$，说明最高的一堆是 3 层。
    
3. **终极锁定**：总数是5，分2堆，最高一堆是3。那唯一的拼法就是 3 和 2！
    
4. **写出标准型**：此时，Jordan 标准型中属于 $\lambda=2$ 的部分，被**唯一确定**为一个 $3 \times 3$ 的 Jordan 块和一个 $2 \times 2$ 的 Jordan 块的直和：$J_3(2) \oplus J_2(2)$。
    
5. **还原 Jordan 链**：在空间中，这对应着两条互不干扰的 Jordan 链：一条长度为 3，一条长度为 2。链条的尽头分别是那两个真正的特征向量。
